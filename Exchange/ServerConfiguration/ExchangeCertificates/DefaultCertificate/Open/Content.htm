To learn how to open an Exchange certificate, proceed with the following steps...
  • Begin by selecting a certificate. In this case we will select the default certificate that was automatically installed on this server when Exchange was installed. 
  • Then select the option 'Open' found in the Actions pane located at the far right side of the Exchange Management Console window. However, In this case, in order to select it, we first need to scroll down.
  • Then, once the option 'Open' is located, select it.
  • When you install Exchange, a self-signed certificate, created and signed by the Exchange server itself, is automatically installed on the server.
