You can assign specific services to your Secure Sockets Layer (SSL) certificate. To learn how to assign services to a certificate, proceed with the following steps...
  • Begin by selecting a certificate. In this case we will select the default certificate that was automatically installed on this server when Exchange was installed. 
  • Then select the option 'Assign Services to Certificate' found in the Actions pane located at the far right side of the Exchange Management Console window. However, In this case, in order to select it, we first need to scroll down.
  • Then, once the option 'Assign Services to Certificate' is located, select it.
  • Next, we need to select the services we want to assign to our certificate. The first service is 'Internet Message Access Protocol'.
  • The next service is 'Post Office Protocol'.
  • Next is 'Simple Mail Transfer Protocol'.
  • And then 'Internet Information Services'.
  • And then finally 'Unified Messaging'.
  • Once the disired service is selected, click 'Assign'.
  • If the 'Unfified Messaging' service is selected, you will then receive a dialog box asking you to confirm the the certificate you are applying unified messaging to.

  • Click 'Yes'.
  • On this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
