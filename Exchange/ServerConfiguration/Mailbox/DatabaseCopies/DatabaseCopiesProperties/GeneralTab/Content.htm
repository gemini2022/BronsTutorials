To learn about the General tab, proceed with the following steps...
  • This read-only field displays the mailbox database
  • This read-only field displays the name of the Mailbox server that hosts the selected database copy.
  • This read-only field displays whether the mailbox database is mounted or dismounted.
  • This read-only field  indicates the number of log files waiting to be copied to the selected database copy. This field is relevant only for passive database copies.
  • This read-only field indicates the number of log files waiting to be replayed into the selected database copy. This field is relevant only for passive database copies.
  • Displays the activation preference number. This is used as part of Active Manager's best copy selection process, and it is used to balance the DAG by redistributing active mailbox databases throughout the DAG via the DAG's PreferenceMoveFrequency property. This property defines the frequency (measured in time) when the Microsoft Exchange Replication service rebalances database copies by performing a lossless switchover that activates the copy with an activation preference number of 1. The value for activation preference is a number equal to or greater than 1, where 1 is at the top of the preference order. The number can't be larger than the number of copies of the mailbox database.
