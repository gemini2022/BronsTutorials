To learn how to switchover a server, proceed with the following steps...
  • Begin first by selecting a server, which in this case will be 'MX1'.
  • After a server has been selected, select the option 'Switchover Server' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • With this first option, 'Automatically choose a target server', Exchange will choose a target server automatically.
  • This option 'Use the specified server as the target for switchover', specifies that a target server needs to be manually chosen.
  • To choose a target sever, begin by first clicking 'Browse'.
  • Then choose a target server in the list. In this case there is no target server to choose from.
  • Click 'Cancel'.
