This wizard helps you associate an external DNS address with one or more Client Access servers. The ExternalURL property for all services is configured when an external Client Access domain is specified. To learn how to configure the external domain used by your Client Access servers, proceed with the following steps...
  • Begin by selecting the option 'Configure External Client Access Domain' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Then enter the domain name you'll use with your external Client Access servers.
  • In this case we will enter 'mail.bronstutorials.com'.
  • Then click 'Add' to select the Client Access servers that will be associated with this ExternalURL.
  • In this case there is only one Client Access server in the list, which is already selected by default.
  • Click 'OK'.
  • After we click 'OK' on the 'Select Client Access Server' dialog box, then the Client Access server will be displayed here.
  • Click 'Configure'.
  • On this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
