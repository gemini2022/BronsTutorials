Use the 'Authentication' tab to control the authentication methods for the Exchange ActiveSync virtual directory. To learn how to do this, proceed with the following steps...
  • Select this check box if you want the mobile device to send the user name and password in clear text.
  • Select this option if you want to ignore client certificate authentication.
  • Select this option if you want to Accept client certificate authentication.
  • Select this option if you want to require client certificate authentication.
