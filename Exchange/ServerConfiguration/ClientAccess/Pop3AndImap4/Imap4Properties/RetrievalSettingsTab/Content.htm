On the 'Retrieval Settings' tab, specify the message and calendar retrieval settings for your IMAP4 clients. To learn how to do this, proceed with the following steps...
  •  First, you can use this dropdown to specify the format of the messages retrieved by the Client Access server.
  • Once the dropdown is selected, you can choose from the following options:
    • Text
    • HTML
    • HTML and alternative text
    • Enriched text
    • Enriched text and alternative text
    • Best body format
    • TNEF
  • Configure this setting so users can use the iCalendar standard for calendar items. The iCalendar standard is a standard for exchanging calendar information.
  • Specify an internal URL for users to use to access their calendar information.
  • Specify an external URL for users to use to access their calendar information.
  • Specify the URL of the Outlook Web App server for users to use to access their calendar information.
  • Specify the Outlook Web App server for custom Outlook Web App calendar items. If you've selected the custom setting for Calendar retrieval, you must specify the Outlook Web App server URL.
