On the 'Binding' tab, specify the IP addresses and TCP ports on the Receive connector that accepts connections from POP3 clients. You can also configure the IP address ranges from which the Receive connector accepts connections from POP3 clients. To learn about the Binding tab, proceed with the following steps...
  • In this section, specify the IP addresses and port numbers for POP3 connections that use Transport Layer Security (TLS) or unencrypted connections. For each entry, you must specify a different set of IP addresses or specify all available IP addresses.
  • Using this list, specify the IP address or IP address range for POP3 connections that use SSL. For each entry, you must specify a different set of IP addresses or specify all available IP addresses.
  • To add a new IP address or port number, begin by clicking 'Add' to open the 'TLS or Unencrypted Connection settings' dialog box.
  • Select this option to use all IP addresses associated with this computer.
  • Select this option if more than one IP address is associated with this computer and you want to bind the Receive connector to a specific IP address.
  • Then, in the text box, type the IP address on which this server accepts connections.
  • In this case the IP address we will enter will be '192.168.1.2'.
  • This text box identifies the TCP port number on which this Receive connector listens for incoming mail. SSL port 110 is the default port used for POP3.
  • Click 'OK'.
  • After the 'OK' button on the 'TLS or Unencrypted Connection settings' dialog box has been clicked, the IP address and the port number will be displayed here.
