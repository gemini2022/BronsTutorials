To learn how to disable a Unified Messaging server after calls are complete, proceed with the following steps...
  • Begin first by selecting a server, which in this case will be 'MX1'.
  • After a server has been selected, select the option 'Disable After Completing Calls' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Once the 'Exchange Management Console' is open, expand the node 'Microsoft Exchange On-Premise'.
