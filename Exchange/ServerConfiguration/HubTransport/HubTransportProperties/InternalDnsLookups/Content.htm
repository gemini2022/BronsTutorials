Use the 'Internal DNS Lookups' tab to specify whether to use the DNS servers that are configured on a network adapter installed on this server or to use specific DNS servers when resolving the addresses of mail servers for internal mail delivery. Internal DNS servers are used to resolve IP addresses for servers inside your organization. To learn about the Internal DNS Lookups tab, proceed with the following steps...
  • To use the DNS servers configured on the network adapters that are installed on this Exchange server, select the option 'Use network card DNS settings'.
  • Then select a network adapter from the dropdown. The server then uses the DNS servers configured on that network adapter. The default setting is All Available IPv4. If you select this setting, the server uses the DNS servers configured on all the adapters on that server.
  • In this case, we will select a specific network adapter from the list.
  • Once a specific network adapter is selected, the IP address of the DNS server that is configured on that network adapter will be displayed here.
  • To have the Exchange server select from a list of manually entered DNS servers to query when resolving an internal server, select the option 'Use these DNS servers'.
  • Then, to add a server to the list, type the IP address of the internal DNS server here.
  • In this case we will enter '192.168.1.2'.
  • Once the IP address of the internal DNS server has been entered, click 'Add'.
  • After clicking 'Add', the IP address of the internal DNS server will be displayed here.
