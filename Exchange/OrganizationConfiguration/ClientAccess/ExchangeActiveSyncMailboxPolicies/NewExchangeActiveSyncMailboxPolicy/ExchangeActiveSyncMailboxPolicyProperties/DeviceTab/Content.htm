Use the Device tab to specify a variety of device-specific settings. All settings that you access on the Device tab of the Exchange ActiveSync policy Properties page are premium features of Exchange ActiveSync. For these features to be implemented on a mobile phone, the mailbox requires an Exchange Enterprise client access license (CAL). To learn about the Device tab, proceed with the following steps...
  • Select this check box to allow storage cards to be accessed from a mobile phone. If this check box isn't selected, storage cards can't be accessed from a mobile phone.
  • Select this check box to allow the mobile phone camera to be used.
  • Select this check box to allow the mobile phone to use a Wi-Fi connection for Internet access. Direct Push isn't supported over Wi-Fi.
  • Select this check box to allow the mobile phone to establish an infrared connection with other devices or computers.
  • Select this check box to allow another device to share the Internet connection of the mobile phone. Internet sharing is frequently used when the device functions as a modem for a laptop or desktop computer.
  • Select this check box to allow the mobile phone to establish a remote desktop connection to another computer.
  • Select this check box to allow the mobile phone to synchronize with a desktop computer through desktop ActiveSync or the Windows Mobile Device Center.
  • Use this dropdown to control the Bluetooth functionality of the mobile phone.
  • Once the dropdown is selected, you can choose from the following options:
    • Disable
    • Handsfree only
    • Allow
