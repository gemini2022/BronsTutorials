Use the General tab to specify the types of mobile phones that can connect to the Exchange server and whether attachments can be synchronized. To learn about the General tab, proceed with the following steps...
  • This unlabeled box at the top of the tab displays the Exchange ActiveSync policy name. You can modify this name.
  • This read-only field displays a True or False status to indicate whether the selected Exchange ActiveSync policy is the default policy.
  • This read-only field displays the last date and time that the Exchange ActiveSync policy was modified.
  • Select this check box to allow mobile phones that can't be provisioned automatically. These mobile phones may be unable to enforce all the Exchange ActiveSync policy settings. By selecting this box, you're allowing these mobile phones to synchronize even though some policy settings may not be applied.
  • Select this check box to force the server to resend the policy to clients at a fixed interval defined in the number of hours between policy refresh events.
  • Then enter the number of hours between policy refresh events here.
  • In this case, we will enter the value of '10'.
