Use the Other tab to specify allowed and blocked applications. All settings that you access on the Other tab of the Exchange ActiveSync policy Properties pages are premium features of Exchange ActiveSync. For these features to be implemented on a mobile phone, the mailbox requires an Exchange Enterprise client access license (CAL). To learn about the Other tab, proceed with the following steps...
  • This section displays a list of all the apllication that are allowed to be installed and run on the mobile phone. Click Add to add an application, and click Delete to remove an application.
  • This section displays a list of all the apllication that are blocked from being installed and run on the mobile phone. Click Add to add an application, and click Delete to remove an application.
