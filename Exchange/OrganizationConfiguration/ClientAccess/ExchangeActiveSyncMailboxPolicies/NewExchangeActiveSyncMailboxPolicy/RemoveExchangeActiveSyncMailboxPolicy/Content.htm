To learn how to remove an Exchange ActiveSync Mailbox Policy, proceed with the following steps...
  • Begin first by selecting an Exchange ActiveSync Mailbox Policy. In this case we will select the new Exchange ActiveSync Mailbox Policy we just created, 'ExchangeActiveSyncMailboxPolicy'.
  • Then select the option 'Remove' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • And then finally press 'Yes' to remove the selected Exchange ActiveSync Mailbox policy.
