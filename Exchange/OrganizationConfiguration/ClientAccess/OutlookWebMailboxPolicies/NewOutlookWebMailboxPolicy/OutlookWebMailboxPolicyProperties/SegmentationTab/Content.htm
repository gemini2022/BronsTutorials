On the Segmentation tab, you can specify the features you want to enable or disable for Outlook Web App users. To learn how to do this, proceed with the following steps...
  • Select a disabled feature in the list, and then click 'Enable' to enable that feature.
  • Select an enabled feature in the list, and then click 'Disable' to disable that feature.
  • The Feature column shows the list of features that are potentially available to Outlook Web App users on a virtual directory.
  • This section displays a description for the selected feature in the list.
