Use the Times tab on the auto attendant Properties sheet to determine the organization's open business hours. During business hours, callers hear the default business hours greeting or a customized greeting and the business hours main menu prompt if the appropriate business hours key mappings are configured on the Key Mapping tab. To learn about the Times tab, proceed with the following steps...
  • Use this dropdown to select a standard schedule.
  • Once the dropdown is selected, you can choose from the following options:
    • Weekdays 8:00 A.M. - 5:00 P.M.
    • Weekdays 9:00 A.M. - 5:00 P.M.
    • Weekdays 9:00 A.M. - 6:00 P.M.
    • Weekdays 8:00 A.M. - 12:00 P.M., 1:00 P.M. - 5:00 P.M.
    • Weekdays 9:00 A.M. - 12:00 P.M., 1:00 P.M. - 6:00 P.M.
    • Always Run
    • Use Custom Schedule
  • Or you can configure a custom time, to do this, click 'Customize'.
  • Then use the 'Select Schedule' dialog box to configure a customized time.
  • Once a customized time is configured, click 'OK'.

  • Use this dropdown to select your time zone. Consider whether the dial plan associated with the auto attendant covers more than one time zone when you set your schedule. By default, the time zone is configured using the local server's system time when the Unified Messaging server role was installed.
  • In this case, we will select the option 'Eastern Time'.
  • Use Holiday Schedule to define days, from 00:00 through 23:59 (12:00 A.M. through 11:59 P.M.), for which your organization will be closed for a holiday. Callers who reach the auto attendant during the times that you specify in the Select Holiday dialog box hear a custom holiday greeting audio file that you define. When you configure the holiday schedule, you must also define the Holiday start date and Holiday end date. To do this, begin first by clicking 'Add' to open the 'Select Holiday' dialog box.
  • Once the 'Select Holiday' dialog box is open, we first need to give our holiday schedule a name.
  • In this case, we will name our holiday schedule 'Christmas Break'.
  • Then we need to click the 'Browse' button to locate a custom holiday greeting audio file that was previously recorded.
  • Next, we need to define the date when the holiday starts. To do this, select the dropdown.
  • And choose from the calandar when the holiday starts.
  • Then we need to define the date when the holiday Ends. To do this, select this dropdown.
  • And choose from the calandar when the holiday ends.
  • Click 'OK'.
