Use the Greetings tab on the auto attendant's Properties sheet to manage recorded greetings for the current UM auto attendant. You can select default greetings or previously recorded custom greetings for business hours and non-business hours. To learn about the Greetings tab, proceed with the following steps...

  • This is the initial greeting played when a caller calls the auto attendant during your organization's business hours. By default, it's a system prompt that says, "Welcome to the Exchange auto attendant." This greeting plays during your organization's business hours. The business and non-business hours are configured on the auto attendant Times tab. You may want to customize this greeting to represent your company, for example, "Thank you for calling Woodgrove Bank."
  • To make this customize greeting, beging first by clicking the 'Modify' button to open the 'Business Hours Greeting' dialog box.
  • Once the 'Business Hours Greeting' dialog box is open, the first option you will see is the option that allows you to enable the default business hours greeting to be played to callers. By default, this option is enabled.
  • Or select this option to enable a custom business hours greeting file to be played to callers.
  • Once this option is selected, click the 'Browse' button to locate a custom business hours greeting file that was previously recorded.
  • In this case there is no custom greeting file created so we will cancel out of this dialog box.

  • This is the initial greeting played when a caller calls the auto attendant during your organization's non-business hours. By default, no business or non-business hours are configured. Therefore, there is no default non-business hours greeting. You can configure the business and non-business hours on the auto attendant Times tab. You may want to customize this greeting to represent your company, for example, "Thank you for calling Woodgrove Bank but we are now closed."

     
  • To make this customize greeting, beging first by clicking the 'Modify' button to open the 'Non-Business Hours Greeting' dialog box.
  • Once the 'Non-Business Hours Greeting' dialog box is open, the first option you will see is the option that allows you to enable the default non-business hours greeting to be played to callers when a business hours schedule has been configured on the auto attendant. By default, this option is enabled.
  • Or select this option to enable a custom non-business hours greeting file to be played to callers. 
  • Once this option is selected, click the 'Browse' button to locate a custom non-business hours greeting file that was previously recorded.
  • In this case there is no custom greeting file created so we will cancel out of this dialog box.

  • When enabled, this optional recording plays immediately after the business or non-business hours greeting. An informational announcement may state the organization's hours of operation, for example, "Our business hours are 8:30 A.M. to 5:30 P.M., Monday through Friday and 8:30 A.M. to 1:00 P.M. on Saturday." An informational announcement can also provide information required for compliance with company policy, for example, "Calls may be monitored for training purposes." If it's important that callers hear the whole informational announcement, it can be marked as uninterruptible.

  • By default, there's no informational announcement configured on UM dial plans or auto attendants. To enable this informational announcement, beging first by clicking the 'Modify' button to open the 'Informational Announcement' dialog box.
  • Once the 'Informational Announcement' dialog box is open, the first option you will see is the option that allows you to disable an informational announcement. By default, this option is configured.
  • Or select this option to to enable an informational announcement to be played to callers.
  • Once this option is selected, click the 'Browse' button to locate a custom informational announcement file previously recorded.
  • Use this option to enable the informational announcement to be interrupted by the caller. This should be enabled if you have long informational announcements. Callers may become frustrated if the informational announcement is long and they can't interrupt the informational announcement to access the options provided by the UM dial plan or auto attendant.
  • In this case there is no informational announcement created so we will cancel out of this dialog box.
  • The business hours main menu prompt for an auto attendant is the list of options callers hear during business hours defined on the Times tab. For example, "For technical support, press or say 1. For corporate offices and administration, press or say 2. For sales, press or say 3."
  • To enable a business hours main menu prompt, beging first by clicking the 'Modify' button to open the 'Business Hours Main Menu Prompt' dialog box.
  • Once the 'Business Hours Main Menu Prompt' dialog box is open, the first option you will see is the option that allows you to enable the default business hours main menu prompt to be played to callers when a business hours schedule has been configured on the auto attendant. By default, this option is enabled.
  • Or select this option to to enable a custom business hours main menu prompt file to be played to callers.
  • Once this option is selected, click the 'Browse' button to locate a previously recorded custom business hours greeting file. The recordings must already have been recorded as .wav files.
  • In this case there is no custom business hours greeting file created so we will cancel out of this dialog box.
  • The non-business hours main menu prompt for an auto attendant is the list of options callers hear during the non-business hours defined on the Times tab. For example, "Your call is very important to us. However, you have reached Woodgrove Bank after normal business hours. If you want to leave a message, please press or say 1 and we will return your call as soon as possible."
  • To enable a non-business hours main menu prompt, beging first by clicking the 'Modify' button to open the 'Non-Business Hours Main Menu Prompt' dialog box.
  • Once the 'Non-Business Hours Main Menu Prompt' dialog box is open, the first option you will see is the option that allows you to enable the default non-business hours main menu prompt to be played to callers when a non-business hours schedule has been configured on the auto attendant. By default, this option is enabled. The recordings must already have been recorded as .wav files.
  • Or select this option to to enable a custom non-business hours main menu prompt file to be played to callers.
  • Once this option is selected, click the 'Browse' button to locate a custom non-business hours greeting file that was previously recorded. The recordings must already have been recorded as .wav files.
  • In this case there is no custom non-business hours greeting file created so we will cancel out of this dialog box.
