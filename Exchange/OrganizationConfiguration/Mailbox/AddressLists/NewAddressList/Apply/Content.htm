Changes that you make to an address list aren't applied to recipients until you apply the list. You can apply changes to address lists immediately or at a scheduled time by using the New Address List wizard or the Edit Address List wizard. To learn how to apply an Address List, proceed with the following steps...
  • Begin first by selecting an Address List. In this case we will select the new Address List we just created, 'AddressList'.
  • Then select the option 'Apply' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Select this option to apply the changes immediately.
  • Or select this option to specify a time to apply the address list.
  • Then select the dropdown.
  • Once the dropdown is selected, a calendar will appear where you can set the date when the address list will be applied.
  • Then you can set the time when the address list will be applied.
  • Not only do you have the option to define when the address list will be applied, but you can also define how long the address list will be applied for. To do this, select the option 'Cancel tasks that are still running after (hours)'.
  • Then enter in how long the address list will run. The default is 8 hours.
  • Click 'Next >' to move on.
  • On this screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
