To learn about the General tab, proceed with the following steps...
  • This unlabeled box at the top of the tab displays the name of the selected managed default folder. You can modify this name.
  • This read-only field displays the description of the managed default folder
  • This read-only field displays the folder type of the managed default folder
  • This read-only field displays the date and time the managed default folder was last modified.
  • Use this box to type a comment that will be displayed to the user in Outlook. For example, to alert users that MRM is enabled on the folder, you could type the following message: "Messages are removed after 100 days." The maximum length of this comment is 255 characters.
  • Select this check box to prevent Outlook users from minimizing the comment that you entered in the preceding box.
