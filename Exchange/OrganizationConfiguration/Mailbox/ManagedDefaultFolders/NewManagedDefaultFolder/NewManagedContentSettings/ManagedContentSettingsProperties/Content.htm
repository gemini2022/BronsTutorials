To learn how to get to the properties of Managed Content Settings, proceed with the following steps...
  • Begin first by expanding a Managed Default Folder that has Managed Content Settings applied to it. In this case we will expand the new Managed Default Folder we just created, 'ManagedDefaultFolder'.
  • Then select its Managed Content Settings.
  • Then select the option 'Properties' found in the Actions pane located at the far right side of the Exchange Management Console window. However, In this case, in order to select it, we first need to scroll down.
  • Then, once the option 'Properties' is located, select it.
