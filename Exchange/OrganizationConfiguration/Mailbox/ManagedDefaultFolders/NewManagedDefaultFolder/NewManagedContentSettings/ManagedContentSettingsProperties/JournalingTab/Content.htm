Use the Journaling tab to specify how to journal (automatically forward) copies of items of specified message types to another location. You can also use this tab to check journaling settings. To learn about the Journaling tab, proceed with the following steps...
  • Select this check box to specify that you want to forward copies of messages. The recipient can be any location that has an SMTP e-mail address, including another Exchange mailbox.
  • In this case, copy of messages will be forwarded to the Administrator.
  • If you want to specify a different recipient to which you want to forward the copies, begin by frist clicking 'Browse'.
  • Then select a recipient to which you want to forward the copies to.
  • Once a recipient is selected, click 'OK'.
  • After selecting the Forward copies to check box, you can type a label that you want to assign to the copy of the message. The label can be used to preserve classification information about messages and to enable automated sorting of journaled messages by the recipient.
  • And after selecting the Forward copies to check box, you can also specify the format of the copies of the messages to be forwarded.
  • The first option, 'Exchange MAPI Message Format (TNEF)',  specifies that messages are forwarded in the format (Transport Neutral Encapsulation Format) that is most commonly used by Microsoft Exchange Server 2007. This is the default choice.
  • And this option, 'Outlook Message Format (*.msg)', specifies that messages are forwarded in the .msg file format that is commonly used by Microsoft Outlook.
