To learn how to remove Managed Content Settings from a Managed Default Folder, proceed with the following steps...
  • Begin first by expanding a Managed Default Folder that has Managed Content Settings applied to it. In this case we will expand the new Managed Default Folder we just created, 'ManagedDefaultFolder'.
  • Then select its Managed Content Settings.
  • Next, select the option 'Remove' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • And then finally press 'Yes' to remove the selected managed content settings.
