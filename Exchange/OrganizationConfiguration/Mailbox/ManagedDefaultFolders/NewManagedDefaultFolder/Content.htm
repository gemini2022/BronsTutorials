A managed default folder is a mailbox folder (such as the Inbox folder) that appears in Office Outlook 2007 by default and to which MRM has been applied. The retention and journaling of messages in managed default folders are controlled by managed content settings that are applied to the folder. To learn how to create a new Managed Default Folder, proceed with the following steps...
  • Begin by selecting the option 'New Managed Default Folder' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Use this box to assign a name to the new managed default folder. This value is used by the administrator as an identifier for the folder. It does not appear in users' mailboxes. (The standard name of the default folder, such as Inbox, appears in users' mailboxes.) This name can be up to 65 characters in length.
  • In this case we will name the new managed default folder 'ManagedDefaultFolder'.
  • Next, we need to select the type of default folder that we want to create. To do this, first select the dropdown.
  • Then select one of the default folders provided in the list.
  • In this case, we will select the default folder, 'All other folders in the mailbox'.
  • Use this box to type a comment that will be displayed to the user in Outlook. For example, to alert users that MRM is enabled on the folder, you could type the following message: "Messages are removed after 100 days." The maximum length of this comment is 255 characters.
  • Select this check box to prevent Outlook users from minimizing the comment that you entered in the preceding box.
  • Press 'New' to create the default folder.
  • On this screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
