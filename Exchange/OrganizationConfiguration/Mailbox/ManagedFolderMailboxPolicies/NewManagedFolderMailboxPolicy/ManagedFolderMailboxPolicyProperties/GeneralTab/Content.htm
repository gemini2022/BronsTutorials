Use the General tab to view the properties of a managed folder mailbox policy and to add or remove associated folders from that policy. To learn about the General tab, proceed with the following steps...
  • This unlabeled box displays the name of the managed folder mailbox policy. You can modify this name.
  • This field displays the last date and time that the managed folder mailbox policy was modified.
  • This list displays the managed folders that are associated with the managed folder mailbox policy.
  • To add a folder to the list, click 'Add'.
  • Then use the 'Select Managed Folder' dialog box to select the folders that will be associated with the managed folder mailbox policy.
  • Once the folders are selected, click 'OK'.
