To learn how to get to the properties of a Managed Folder Mailbox Policy, proceed with the following steps...
  • Begin first by selecting a Managed Folder Mailbox Policy. In this case we will select the new Managed Folder Mailbox Policy we just created, 'MailboxPolicy'.
  • Then select the option 'Properties' found in the Actions pane located at the far right side of the Exchange Management Console window.
