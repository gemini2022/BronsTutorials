To learn how to get to the properties of a Database Availability Group, proceed with the following steps...
  • Begin first by selecting a Database Availability Group. In this case we will select the new Database Availability Group we just created, 'DAG1'.
  • Then select the option 'Properties' found in the Actions pane located at the far right side of the Exchange Management Console window. However, In this case, in order to select it, we first need to scroll down.
  • Then, once the option 'Properties' is located, select it.
