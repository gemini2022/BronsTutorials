To learn how to remove a Database Availability Group, proceed with the following steps...
  • Begin first by selecting a Database Availability Group. In this case we will select the new Database Availability Group we just created, 'DAG1'.
  • Then select the option 'Remove' found in the Actions pane located at the far right side of the Exchange Management Console window. However, In this case, in order to select it, we first need to scroll down.
  • Then, once the option 'Remove' is located, select it.
  • Once the 'Exchange Management Console' is open, expand the node 'Microsoft Exchange On-Premise'.
