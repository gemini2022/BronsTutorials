You can use sharing policies to control how users in your organization can share calendar and contact information with users outside your Exchange organization. Sharing policies support the sharing of calendar and contact information with external federated organizations, external non-federated organizations, and individuals with Internet access. To learn how to create a new Sharing Policy, proceed with the following steps...
  • Begin by selecting the option 'New Sharing Policy' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Use this box to type a name for the new sharing policy.
  • In this case, the name we will enter for the new sharing policy will be 'SharingPolicy2'.
  • Next, click this button to open the 'Add Action to Sharing Policy Domain' dialog box. Use this dialog box to specify the domains and actions that this sharing policy applies to.
  • Once the 'Add Action to Sharing Policy Domain' dialog box is open, begin by typing the fully qualified domain name of an external Exchange organization, for example, Contoso.com.
  • In this case, we will enter '*'. This means that this policy will apply to any domain.
  • Next, we need to to define the type of sharing level we want to enforce. To do this, begin by selecting the dropdown.
  • In this case we will select the option 'Contacts sharing'.
  • Press 'OK'.
  • After we click 'OK' on the 'Add Action to Sharing Policy Domain' dialog box, then the domain and it's associated sharing action that we defined will be displayed here.
  • Select this check box to enable the sharing of calendar and contact information with recipients in the external domains that you specified in the policy. If you clear this check box, mailboxes assigned this policy won't be able to share calendar and contact information with the external domains or users specified in the policy. Additionally, existing shared calendar and contacts folders will no longer be shared with the external domains or users specified in the policy.
  • Press 'Next >'.
  • On this screen, we first need to choose the mailboxes to which we want to apply this sharing policy to. To do this, begin by clicking 'Add' to open the 'Select Mailbox' dialog box.
  • Once the 'Select Mailbox' dialog box is open, select the mailbox you want to use with this policy. In this case there is only one mailbox, which is already selected by default.
  • Press 'OK'.
  • After we click 'OK' on the 'Select Mailbox' dialog box, then the mailbox will be displayed here.
  • Click 'Next >'.
  • On this screen, review the 'Configuration Summary' and verify that the information you entered is correct. If you are not satisfied with any of the information, you may go back and make any changes necessary.
  • If the information in the 'Configuration Summary' is correct, press 'New'.
  • And then finally, on this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
