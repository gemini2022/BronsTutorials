Use the General tab to view or modify the sharing policy information. To learn how to do this, proceed with the following steps...
  • This unlabeled box lists the name of the sharing policy. You can modify this field.
  • This field displays the most recent date and time that the sharing policy settings were modified.
  • This box displays a list of the domains and actions that this sharing policy applies to.
  • To add a domain and action to the list, click the 'Add' button.
  • Use this box to type the fully qualified domain name (FQDN) of an external Exchange organization, for example Contoso.com.
  • Then use this dropdown to select a sharing level you want to enforce.
  • Since no information was entered, we will cancel out of this dialog box.
  • Select this check box to enable the sharing of calendar and contact information with recipients in the external domains that you specified in the policy. If you clear this check box, mailboxes assigned this policy won't be able to share calendar and contact information with the external domains or users specified in the policy. Additionally, existing shared calendar and contacts folders will no longer be shared with the external domains or users specified in the policy.
