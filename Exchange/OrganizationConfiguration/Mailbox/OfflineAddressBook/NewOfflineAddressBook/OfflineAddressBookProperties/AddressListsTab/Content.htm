Use the Address Lists tab to select the address lists you want to include in the OAB. If you want to include a global address list (GAL) other than the default GAL, you must use the Shell. To learn about the Address Lists tab, proceed with the following steps...
  • Select this check box to include the default GAL in the OAB.
  • Select this check box to add address lists to or remove address lists from the OAB.
  • The address list that are associated with the OAB will be displayed here.
  • You have the option of adding additional address lists if needed. To do this, begin by first clicking the 'Add' button to open the 'Select Address list' dialog box.
  • Then select an address list from the list.
  • In this case we will cancel out of the 'Select Address list' dialog box, because the address list we want has already been selected.
