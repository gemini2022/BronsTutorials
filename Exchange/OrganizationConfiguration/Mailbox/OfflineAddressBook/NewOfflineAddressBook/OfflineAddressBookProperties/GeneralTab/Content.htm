Use the General tab to view OAB properties and to set the update interval for the OAB. To learn how to do this, proceed with the following steps...
  • This unlabeled box at the top of the tab displays the OAB name. You can modify this name.
  • This read-only field displays the OAB generation server. The OAB generation server is the Mailbox server on which the OABs are generated. If you want to specify a different generation server, use the Move-OfflineAddressBook cmdlet with the Server parameter.
  • This read-only field displays a True or False status to indicate whether the selected OAB is the default OAB. If this isn't the default OAB, and you want to set it as the default, right-click the OAB in the result pane, and then click Set as Default.
  • This read-only field displays the last date and time that the OAB was modified.
  • This dropdown displays the time and interval for the regularly scheduled update.
  • Once the dropdown is selected, you can choose from the following options:
    • Run daily at 1:00 A.M.
    • Run daily at 2:00 A.M.
    • Run daily at 3:00 A.M.
    • Run daily at 4:00 A.M.
    • Run daily at 5:00 A.M.
    • Never Run
    • Use Custom Schedule
  • Or you can configure a custom time, to do this, click 'Customize'.
  • Then use the 'Select Schedule' dialog box to configure a customized time.
  • Once a customized time is configured, click 'OK'.
