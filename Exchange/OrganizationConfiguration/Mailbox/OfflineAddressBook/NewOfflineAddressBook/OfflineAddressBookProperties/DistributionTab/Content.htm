Use the Distribution tab to specify the client support and OAB distribution points for the OAB. To learn how to do this, proceed with the following steps...
  • Fist, you have the option of selecting the OAB version that will be generated for the version of Outlook that is used by your Exchange organization. The first option is 'Outlook 98 SP1 or earlier (Version 2)'.
  • The next option is 'Outlook 98 SP2 or later (Version 3)'.
  • And the last option is 'Outlook 2003 SP2 or later (Version 4)'.
  • Select this check box to enable Web-based distribution. Web-based distribution is the distribution method by which Outlook 2007 or later clients that are working offline or through a dial-up connection access the OAB. With Web-based distribution, a Client Access server will contain an OAB virtual directory for Web distribution purposes.
  • The virtual directory or directories from which you want to distribute the OAB will be listed here.
  • You have the option of adding additional virtual directories if needed. To do this, begin by first clicking the 'Add' button to open the 'Select OAB Virtual Directory' dialog box.
  • Then select a virtual dirctory from the list.
  • In this case we will cancel out of the 'Select OAB Virtual Directory' dialog box, because the virtual directory that is listed has already been selected.
  • And then finally, select this check box to enable public folder distribution. Public folder distribution is the distribution method by which Outlook 2003 or earlier clients that are working offline or through a dial-up connection access OABs.
