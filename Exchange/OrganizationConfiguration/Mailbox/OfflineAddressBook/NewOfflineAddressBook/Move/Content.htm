Offline address book (OAB) generation is the process by which Exchange creates and updates the OAB. When OAB generation occurs, Exchange generates new OAB files, compresses the files, and then places the files on a local share. In this tutorial we are going to move OAB generation to another server. To learn how to do this, proceed with the following steps...
  • Begin first by selecting an Offline Address Book. In this case we will select the new Offline Address Book we just created, 'OfflineAddressBook2'.
  • Then select the option 'Move' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • This read-only field displays the name of the offline address book for which you are going to perform the move operation on.
  • This field displays the current generation server of the selected offline address book.
  • Begin the move proccess by first clicking the 'Browse' button.
  • Then select the server to which you want to move the OAB generation process. In this case, there is only one mailbox server available and by default is already selected.
  • Press 'OK'.
  • After the new generation server has been defined, press 'Move'.
  • The offline address book can't be moved because it is already generated on the target server.
