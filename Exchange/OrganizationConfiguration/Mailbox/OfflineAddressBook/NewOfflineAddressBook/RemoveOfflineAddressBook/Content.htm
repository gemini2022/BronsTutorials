To learn how to remove an Offline Address Book, proceed with the following steps...
  • Begin first by selecting an Offline Address Book. In this case we will select the new Offline Address Book we just created, 'OfflineAddressBook2'.
  • Then select the option 'Remove' found in the Actions pane located at the far right side of the Exchange Management Console window. However, In this case, in order to select it, we first need to scroll down.
  • Then, once the option 'Remove' is located, select it.
  • And then finally press 'Yes' to remove the selected offline address book.
