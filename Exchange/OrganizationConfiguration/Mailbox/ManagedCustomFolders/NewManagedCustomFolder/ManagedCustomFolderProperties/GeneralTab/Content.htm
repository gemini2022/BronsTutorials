To learn about the General tab, proceed with the following steps...
  • This unlabeled box at the top of the tab displays the name of the selected managed custom folder. You can modify this name.
  • This read-only field displays the description of the managed custom folder
  • This read-only field displays the date and time the managed custom folder was last modified.
  • This is the name that appears in the user's mailbox.
  • Enable this option if you want to set a storage limit to the managed custom folder.
  • Then enter the storage limit.
  • In this case, we will enter '1000' KB.
  • Use this box to type a comment that will be displayed to the user in Outlook.
  • Select this check box to prevent Outlook users from minimizing the comment that you entered in the preceding box.
