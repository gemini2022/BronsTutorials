To learn about the General tab, proceed with the following steps...
  • This unlabeled box at the top of the tab displays the name of the selected managed content settings. You can modify this name.
  • This field displays the most recent date and time that the managed content settings were modified.
  • This read-only box displays the types of messages that are controlled by the managed content settings.
  • Select this check box to specify that items have a retention period.
  • In this box to the right of the check box, type the number of days in the retention period. (The range of values is from 1 through 24,855 days.)
  • After selecting Length of retention period (days), you can use this dropdown to select when the retention period starts.
  • Once the dropdown is selected, you can choose from the following options:
    • When delivered, end date for calendar and recurring tasks(This is the default choice.)
    • When item is moved to the folder
  • And after selecting Length of retention period (days) check box, you can also use this dropdown to select what should happen to an item when the retention period is exceeded.
  • Once the dropdown is selected, you can choose from the following options:
    • Move to the Deleted Items folder (This is the default choice.)
    • Move to a managed custom folder
    • Delete and allow recovery
    • Permanently delete
    • Mark as past retention limit (This option makes items render as gray, strikethrough text in Microsoft Outlook.)
  • In this case, we will select the option 'Move to a Managed Custom Folder'.
  • After selecting the option 'Move to a managed custom folder' we next need to select the managed custom folder to which items that are past their retention settings will be moved to. To do this, first click the 'Browse' button to open the 'Select Managed Folder' dialog box.
  • Once the 'Select Managed Folder' dialog box is open, select the managed custom folder you want to use. In this case there is only one managed custom folder, which is already selected by default.
  • Click 'OK'.
