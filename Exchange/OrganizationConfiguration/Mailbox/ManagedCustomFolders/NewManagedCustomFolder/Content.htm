A managed custom folder is a managed folder that’s created by an Exchange administrator and placed in a user's mailbox for messaging records management (MRM) purposes. The retention and journaling of messages in managed custom folders are controlled by managed content settings that are applied to the folder. To learn how to create a new Managed Custom Folder, proceed with the following steps...
  • Begin by selecting the option 'New Managed Custom Folder' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Use this box to assign a name to the new managed custom folder. This value is used by the administrator as an identifier for the folder. It does not appear in users' mailboxes.
  • In this case we will name the new managed custom folder 'ManagedCustomFolder'.
  • This is the name that does appear in the user's mailbox.
  • Enable this option if you want to set a storage limit to the managed custom folder.
  • Use this box to type a comment that will be displayed to the user in Outlook.
  • Select this check box to prevent Outlook users from minimizing the comment that you entered in the preceding box.
  • Press 'New' to create the custom folder.
  • On this screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
