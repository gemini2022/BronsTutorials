To learn how to remove a Managed Custom Folder, proceed with the following steps...
  • Begin first by selecting a Managed Custom Folder. In this case we will select the new Managed Custom Folder we just created, 'ManagedCustomFolder'.
  • Then select the option 'Remove' found in the Actions pane located at the far right side of the Exchange Management Console window. However, In this case, in order to select it, we first need to scroll down.
  • Then, once the option 'Remove' is located, select it.
  • And then finally press 'Yes' to remove the selected managed custom folder.
