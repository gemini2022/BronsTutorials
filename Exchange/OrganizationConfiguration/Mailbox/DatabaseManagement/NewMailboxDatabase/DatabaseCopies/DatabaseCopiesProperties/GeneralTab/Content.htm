To learn about the General tab, proceed with the following steps...
  • Displays the name of the selected database.
  • Displays the name of the Mailbox server that hosts the selected database copy.
  • Displays the current status of the selected database copy.
  • Indicates the number of log files waiting to be copied to the selected database copy. This field is relevant only for passive database copies.
  • Indicates the number of log files waiting to be replayed into the selected database copy. This field is relevant only for passive database copies.
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
