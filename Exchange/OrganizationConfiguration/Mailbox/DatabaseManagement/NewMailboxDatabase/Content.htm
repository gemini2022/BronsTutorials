You can create a mailbox database, which is a unit of granularity where mailboxes are created and stored. A mailbox database is stored as an Exchange database (.edb) file. To learn how to create a new mailbox database, proceed with the following steps...
  • Begin by selecting the option 'New Mailbox Database' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Use this box to type a name for the new mailbox database. The name can contain up to 64 characters but can't include the following characters: \ / " = , ;.
  • In this case, we will name the new mailbox database 'MailboxDatabase2'.
  • Next, click the 'Browse' button to locate the server on which to create the database.
  • Select the server, which in this case there is only one.
  • Once a server is selected, press 'OK'.
  • Now that we have a name for the new mailbox database and the location is defined where it will be placed, we can move on. Press 'Next >'.
  • On this next screen, this first field displays the default path to the database file. You can change the location by typing a new path.
  • This next field displays the default path to the log folder. You can change the location by typing a new path.
  • Select this check box if you want to mount this database. Mounting a database puts it online, making its data available to users.
  • Press 'Next >'.
  • On this screen, review the 'Configuration Summary' and verify that the information you entered is correct. If you are not satisfied with any of the information, you may go back and make any changes necessary.
  • If the information in the 'Configuration Summary' is correct, press 'New'.
  • And then finally, on this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
