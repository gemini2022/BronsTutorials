Use the General tab to view status about the mailbox database, including the mailbox database path, last backup, and mailbox database status. To learn how to do this, proceed with the following steps...
  • This unlabeled box at the top of the tab displays the name of the mailbox database. You can modify this name.
  • This read-only field displays the full path to the Exchange database (.edb) file for the selected mailbox database. To view the entire path, you may have to click the path and use the Right Arrow key. You can't use this field to change the path. To change the location of the database files, close Properties, right-click the database, and then click Move Database Path. You can also change the location by using the Move-DatabasePath cmdlet.
  • This read-only field displays the date and time of the last complete backup of the mailbox database.
  • This read-only field displays the date and time of the last incremental backup of the mailbox database.
  • This read-only field displays whether the mailbox database is mounted or dismounted.
  • This read-only field displays which server the database is mounted on.
  • This read-only field displays the master server for the mailbox database. The Mailbox server that hosts the active copy of a database is referred to as the mailbox database master.
  • This read-only field displays the type of mailbox database master.
  • This read-only field displays the date and time the database was last modified.
  • This read-only field displays the other servers that have a copy of this database.
