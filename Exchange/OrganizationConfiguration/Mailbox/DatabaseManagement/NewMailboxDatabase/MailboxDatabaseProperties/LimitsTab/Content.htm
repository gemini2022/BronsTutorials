Use the Limits tab to specify the storage limits, the warning message interval, and the deletion settings for a mailbox database. To learn how to do this, proceed with the following steps...
  • Select this check box to automatically warn mailbox users that their mailbox is approaching its storage limit. To specify the storage limit, select the check box, and then specify in kilobytes (KB) how much content can be stored in the mailbox before a warning e-mail message is sent to the mailbox users. You can enter a value from 0 through 2,097,151 megabytes (MB) (2.0 terabytes).
  • Select this check box to prevent users from sending new e-mail messages after the size of their mailbox reaches the specified limit. To specify this limit, select the check box, and then type the size of the mailbox in MB at which you want to prohibit the sending of new e-mail messages and notify the user. You can enter a value from 0 through 2,097,151 MB (2.0 terabytes).
  • Select this check box to prevent users from sending and receiving e-mail messages after their mailbox size reaches the specified limit. To specify this limit, select the check box, and then type the size of the mailbox in MB at which you want to prohibit the sending and receiving of e-mail messages and notify the user. You can enter a value from 0 through 2,097,151 MB (2.0 terabytes).
  • Use this dropdown to specify the time at which mailboxes are scanned for compliance with the storage limits that you set.
  • Once the dropdown is selected, you can choose from the following options:
    • Run daily at midnight
    • Run daily at 1:00 A.M.
    • Run daily at 2:00 A.M.
    • Use Custom Schedule
  • Or you can configure a custom time, to do this, click 'Customize'.
  • Then use the 'Select Schedule' dialog box to configure a customized time.
  • Once a customized time is configured, click 'OK'.
  • Use this box to set the number of days that deleted items are retained in a mailbox. You can enter a value from 0 through 24,855 days.
  • Use this box to set the number of days that deleted mailboxes are retained. You can enter a value from 0 through 24,855 days.
  • Select this check box to prevent mailboxes and e-mail messages from being deleted until after the mailbox database has been backed up.
