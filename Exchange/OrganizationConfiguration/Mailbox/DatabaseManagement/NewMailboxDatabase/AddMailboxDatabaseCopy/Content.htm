When you add a copy of a mailbox database, continuous replication is automatically enabled between the existing database and the database copy. Database copies are automatically assigned an identity in the format of <DatabaseName>\<HostMailboxServerName>. For example, a copy of a database named DB1 that's hosted on a server named MBX3 would be named DB1\MBX3. To learn how to add a mailbox database copy, proceed with the following steps...
  • Begin first by selecting a mailbox database. In this case we will select the new mailbox database we just created, 'MailboxDatabase2'.
  • Then select the option 'Add Mailbox Database Copy' found in the Actions pane located at the far right side of the Exchange Management Console window. However, In this case, in order to select it, we first need to scroll down.
  • Then, once the option 'Add Mailbox Database Copy' is located, select it.
  • This read-only field displays the name of the database for which you are going to make a copy.
  • Next, click 'Browse' to open the 'Select Mailbox Server' dialog box. This dialog box will be used to select the Mailbox server that will host the copy of the mailbox database.
  • In this case, there are no other mailbox servers available to store a copy of the mailbox database on, but if there were, they would be listed here.
  • Because no selection can be made, we will just press 'Cancel'.
  • This next field is used to specify the activation preference number for the database copy. The activation preference number is used as part of Active Manager's best copy selection process and to redistribute active mailbox databases throughout the DAG when using the RedistributeActiveDatabases.ps1 script. The value for the activation preference is a number equal to or greater than 1, where 1 is at the top of the preference order. The position number cannot be larger than the number of copies of the mailbox database.
  • This field displays all the servers that already have a copy of the mailbox database.
  • Finally, click 'Add' to add the copy of the mailbox database.
