Use the Limits tab to specify the storage limits, warning message interval, deletion settings, and age limits for all public folders in the selected public folder database. To learn how to do this, proceed with the following steps...
  • Select this check box to automatically warn public folder owners that the public folder is approaching its storage limit. Then specify in megabytes (MB) how much content can be stored in the public folder before a warning e-mail message is sent to the folder's owner. You can enter a value from 0 through 2,097,151 MB (2.0 terabytes)
  • Select this check box to prevent users from sending new e-mail messages after the size of their mailbox reaches the specified limit. To specify this limit, select the check box, and then type the size of the mailbox in MB at which you want to prohibit the sending of new e-mail messages and notify the user. You can enter a value from 0 through 2,097,151 MB (2.0 terabytes).
  • Select this check box to limit the maximum size of items that users can post to the public folders in the database. Then specify the maximum size of items in megabytes (MB) that users can post to the public folders. You can enter a value from 0 through 2,097,151 MB (2.0 terabytes)
  • Use this dropdown to display the interval at which you want warning messages to be generated.
  • Once the dropdown is selected, you can choose from the following options:
    • Run daily at midnight
    • Run daily at 1:00 A.M.
    • Run daily at 2:00 A.M.
    • Use Custom Schedule
  • Or you can configure a custom time, to do this, click 'Customize'.
  • Then use the 'Select Schedule' dialog box to configure a customized time.
  • Once a customized time is configured, click 'OK'.
  • Use this box to set the numbers of days that deleted items are retained in a public folder. You can enter a value between 0 and 24,855 days.
  • Select this check box to prevent items from being permanently deleted until after the public folder database is backed up.
  • Select this check box to limit the age of all folders in this public folder database. Use the text box to specify the age limit in days. You can enter a value between 0 and 24,855 days.
