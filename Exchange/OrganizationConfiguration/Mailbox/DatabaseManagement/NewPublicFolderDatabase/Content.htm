Public folders are designed for shared access and provide an easy and effective way to collect, organize, and share information with other people in your workgroup or organization. A public folder database is a database that stores public folders, indexes their contents, and assists in the replication of the folders with other servers. A public folder database is stored as an Exchange database (.edb) file. To learn how to create a new public folder database, proceed with the following steps...
  • Begin by selecting the option 'New Public Folder Database' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Use this field to type the name of the new public folder database.
  • In this case, we will name our new public folder database 'PublicFolderDatabase'.
  • This next field specifies what server the public folder database will be located. To specify a location, click 'Browse' to open the 'Select Mailbox Server' dialog box. This dialog box will be used to select the server where you want to create the public folder database.
  • Once the 'Select Mailbox Server' dialog box is open, select the server where you want to create the public folder database. In this case there is only one server, which is already selected by default.
  • Press 'OK'.
  • Now that the server where the public folder database will be created is defined, we can move on.
  • Press 'Next'.
  • On this next screen, the first field is automatically populated with the default path to the database file. To specify a different location or file name, type the location in the box.
  • This next field is also automatically populated. This field contains the default path to the database log files. To specify a different location, type the location in the box.
  • Select this check box to mount the database immediately after it is created.
  • Press 'Next'.
  • On this screen, review the 'Configuration Summary' and verify that the information you entered is correct. If you are not satisfied with any of the information, you may go back and make any changes necessary.
  • If the information in the 'Configuration Summary' is correct, press 'New'.
  • And then finally, on this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
