To learn how to remove a public folder database, proceed with the following steps...
  • Begin first by selecting a public folder database. In this case we will select the new public folder database we just created, 'PublicFolderDatabase'.
  • Then select the option 'Remove' found in the Actions pane located at the far right side of the Exchange Management Console window. However, In this case, in order to select it, we first need to scroll down.
  • Then, once the option 'Remove' is located, select it.
  • After the 'Remove' option is selected, a dialog box will appear asking if you ae sure if you want to remove the selected public folder database. Press 'Yes'.
  • Another dialog box may appear if you are removing the last public folder database in the organization. If this is the case, the dialog box will inform you that all contents of the database will be lost and only users running Outlook 2007 or later will be able to connect to your Exchange organization.

  • Press 'Yes'.
  • After the public folder database is removed, yet another dialog box will appear informing you that you must remove all database files associated with this database.
  • Press 'OK'.
