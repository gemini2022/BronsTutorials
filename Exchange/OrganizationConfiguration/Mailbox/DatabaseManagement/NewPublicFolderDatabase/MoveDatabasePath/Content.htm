To learn how to move a public folder database, proceed with the following steps...
  • Begin first by selecting a public folder database. In this case we will select the new public folder database we just created, 'PublicFolderDatabase'.
  • Then select the option 'Move Database Path' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • This read-only field displays the name of the database for which you are going to perform the move operation on.
  • Next, you can change the location for the database file path by editing the 'Database file path' text field.
  • And you can change the location for the log folder path by editing the 'Log folder path' text field.
  • Once the paths have been updated, press 'Move'.
  • After you press 'Move', a dialog box will appear informing you that the public folder database will be temporarily dismounted. Press 'Yes' to perform the move operation.
