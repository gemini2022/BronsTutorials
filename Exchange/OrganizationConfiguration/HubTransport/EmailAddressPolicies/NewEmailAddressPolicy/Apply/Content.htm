E-mail address policies generate the primary and secondary e-mail addresses for your recipients (which include users, contacts, and groups) so they can receive and send e-mail. After you create an e-mail address policy, you need to apply it to the intended recipients for the policy to take effect. To learn how to apply an E-mail Address Policy, proceed with the following steps...
  • Begin first by selecting an E-mail Address Policy. In this case we will select the new E-mail Address Policy we just created, 'Email AddressPolicy'.
  • Then select the option 'Apply' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Select this option to apply the changes immediately.
  • Or select this option to specify a time to apply the e-mail policy.
  • Then select the dropdown.
  • Once the dropdown is selected, a calendar will appear where you can set the date when the e-mail policy will be applied.
  • Then you can set the time when the e-mail policy will be applied.
  • In this case, we will select the option 'Immediately', so that the e-mail address policy will be applied immediately.
  • Not only do you have the option to define when the e-mail policy will be applied, but you can also define how long the policy will be applied for. To do this, select the option 'Cancel tasks that are still running after (hours)'.
  • Then enter in how long the e-mail policy will run for. The default is 8 hours.
  • Click 'Next >' to move on.
  • On this screen, review the 'Configuration Summary' and verify that the information you entered is correct. If you are not satisfied with any of the information, you may go back and make any changes necessary.
  • If the information in the 'Configuration Summary' is correct, press 'Apply'.
  • And then finally, on this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
