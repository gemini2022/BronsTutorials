Use the General tab to specify how Exchange handles the e-mail messages that it accepts for this domain. To learn about the General tab, proceed with the following steps...
  • This unlabeled box at the top of the tab displays the Accepted Domain name. You can modify this name.
  • This is a display name for the accepted domain and doesn't have to match the actual SMTP domain.
  • This read-only field displays a True or False status to indicate whether the selected Accepted Domain is the default Accepted Domain.
  • This read-only field displays the last date and time that the Accepted Domain was modified.
  • Select this option if this domain is used for e-mail addresses of the recipients in your organization.
  • To specify that e-mail messages are either delivered to recipients in your organization or relayed to a server outside your Exchange organization but still under the authority of your company or IT department, select this option.
  • To specify that e-mail messages sent to recipients in this domain are relayed to an e-mail server outside your organization, select this option.
