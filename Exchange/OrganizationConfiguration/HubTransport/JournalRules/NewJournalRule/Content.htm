Journaling is the ability to record all communications, including email communications, in an organization for use in the organization's email retention or archival strategy. To meet an increasing number of regulatory and compliance requirements, many organizations must maintain records of communications that occur when employees perform daily business tasks. Data archiving refers to backing up the data, removing it from its native environment, and storing it elsewhere, therefore reducing the strain of data storage. You can use Exchange journaling as a tool in your email retention or archival strategy. To learn how to create a new Journal Rule, proceed with the following steps...
  • Begin by selecting the option 'New Journal Rule' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Type a name for the journal rule.
  • In this case we will name our new journal rule 'JournalRule'.
  • Next, we need to select the recipient that will receive the journal reports. To do this, begin by first clicking on 'Browse' to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, select the recipient that will receive the journal reports. In this case there is only one recipient, the Administrator, which is already selected by default.
  • Press 'OK'.
  • Next, we need to select the scope to which the journal rule should be applied. This first scope 'Global - all messages', processes all e-mail messages that pass through a Hub Transport server. These include messages that may have already been processed by internal rules and external rules.
  • This next scope, 'Internal - internal messages only', processes e-mail messages sent and received by recipients in the Exchange organization.
  • And then finally this last scope, 'External - messages with an external sender or recipient', processes e-mail messages sent to recipients or sent from senders outside the Exchange organization.
  • If you want to journal messages sent to or from a specific recipient, select the option 'Journal messages for recipient'.
  • Then click 'Browse' to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, select the mailbox, contact, or distribution group that you want to journal. In this case there is only one recipient, and that recipient has already been chosen as the recipient that will receive the journal reports. We can't have the recipient that will receive the journal reports and the recipient that we want to journal to be the same recipient.
  • Therefore, in this case we need to click 'Cancel'.
  • Then disable the option to specify a particular recipient.
  • Next, journal rules are enabled by default. To create the rule in a disabled state, clear the check box.
  • Click 'New' to create the new journal rule.
  • On this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
