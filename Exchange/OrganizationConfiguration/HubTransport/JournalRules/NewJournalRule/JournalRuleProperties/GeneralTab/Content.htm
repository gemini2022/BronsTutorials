Use the General tab to view or configure journal rule properties. To learn how to do this, proceed with the following steps...
  • This field shows the name of the journal rule. You can change the journal rule name. The length of the name can't exceed 64 characters.
  • This field displays rule status as enabled or disabled. You can't modify this field.
  • This field displays the timestamp when the rule was last modified or the rule creation time if the rule has never been modified. You can't modify this field.
  • This field shows the recipient that will receive the journal reports.
  • To change the e-mail address that receives journal reports, click 'Browse'.
  • In the Select Recipient dialog box, select the mailbox, contact, or distribution group to send journal reports to.
  • Since we already have the recipient we want, we will cancel out of this dialog box.
  • These fields specify whether the journal rule is applied to internal recipients and senders only, to external recipients and senders only, or to both. This option applies to all messages that pass through a Hub Transport server.
  • This option applies to messages sent and received by recipients in the Exchange organization.
  • And this option applies to messages sent or received by recipients outside the Exchange organization.
  • This field specifies the mailbox, contact, or distribution group that you want to journal. To target a specific recipient or distribution group, first select the 'Journal messages for recipient' check box.
  • Then click 'Browse' to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, select the mailbox, contact, or distribution group that you want to journal. In this case, we will select 'Distribution Group'.
  • Once a mailbox, contact, or distribution group that you want to journal is selected, click 'OK'.
  • Once you click 'OK' on the 'Select Recipient' dialog box, the mailbox, contact, or distribution group that you selected will be displayed here.
