To learn how to edit a Transport Rule, proceed with the following steps...
  • Begin first by selecting a Transport Rule. In this case we will select the new Transport Rule we just created, 'TransportRule'.
  • Then select the option 'Edit Rule' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • To begin, we can start by editing the name of the transport rule if required.
  • Or we can edit the comment if also required.
  • Click 'Next >'.
  • On this screen we can modify the conditions listed in step 1 by either clearing the check boxes for existing selected conditions or by selecting new conditions as required.
  • Then, any value can be modified in step 2 if required.
  • Click 'Next >'.
  • On this next screen, we can modify the rule actions listed in step 1 by either clearing the check boxes for existing actions or by selecting a new action as required.
  • Then, any value can be modified in step 2 if required.
  • Click 'Next >'.
  • Then on this screen, we can modify the exceptions listed in step 1 by either clearing the check boxes for existing selected exceptions or by selecting new exceptions as required.
  • Then, any value can be modified in step 2 if required.
  • Click 'Next >'.
  • On this screen, review the 'Configuration Summary' and verify that the information you entered is correct. If you are not satisfied with any of the information, you may go back and make any changes necessary.
  • If the information in the 'Configuration Summary' is correct, press 'Update'.
  • And then finally, on this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
