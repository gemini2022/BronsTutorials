Use the Message Format tab to specify message policy, format, and character sets for the messages sent to this remote domain. To learn about the Message Format tab, proceed with the following steps...
  • To allow messages that are automatic replies from client e-mail programs in your organization, select this option.
  • To allow messages that are auto-forwarded by client e-mail programs in your organization, select this option.
  • To allow read receipts from client software in your organization to the remote domain, select this option.
  • To allow non-delivery reports (NDRs) from your organization, select this option.
  • To have the sender's display name appear on messages, select this option. We recommend that you leave this option selected.
  • To allow line wrap in message text for outgoing messages, select this option.
  • And then in the text box, type the line-wrap size, from 0 through 132. To set the value to unlimited, leave the field blank. The default value is unlimited (blank).
  • In this case, we will enter the value of '50'.
  • To always send messages that use Exchange rich-text format, select this text box.
  • To never send messages that use Exchange rich-text format, select this check box.
  • To send e-mail messages that use the Exchange rich-text settings specified by the Outlook user, select this check box.
  • To identify a MIME character set, select the character encoding set from the dropdown
  • Once the dropdown is selected, you can choose from these options.
  • To identify a non-MIME character set, select the character encoding set from this dropdown.
  • Once the dropdown is selected, you can choose from these options.
