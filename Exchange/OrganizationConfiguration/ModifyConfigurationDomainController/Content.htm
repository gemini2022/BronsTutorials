The 'Modify Configuration Domain Controller' feature allows you to specify a domain controller that will be used to read and write to Active Directory when any server or organization configuration is modified. To learn how to configure this, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then open the 'Exchange Management Console' by selecting the 'Exchange Management Console' icon.
  • Once the 'Exchange Management Console' is open, expand the node 'Microsoft Exchange On-Premise'.
  • Then select 'Organization Configuration'.
  • After 'Organization Configuration' is selected, select the option 'Modify Configuration Domain Controller' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • This first option specifies that the default domain controller is to be used. The default domain controller is the domain controller to which the computer is currently connected.
  • Or you can specify a domain controller instead of using a default one. To do this, select the option 'Specify a domain controller'.
  • Then click 'Browse' to open the 'Select Domain' dialog box. Use this dialog box to select a domain in your Active Directory forest. You must select a domain before you can select a domain controller.
  • Once the 'Select Domain' dialog box is open, select a domain, which in this case there is only one domain.
  • After a selection has been made, press 'OK'.
  • Next, click 'Browse' to open the 'Select Domain Controller' dialog box.
  • Use this dialog box to select the domain controller you want to use when modifying server or organization configuration. In this case there is only one domain controller.
  • After a selection has been made, press 'OK'.
  • Press 'OK'.
