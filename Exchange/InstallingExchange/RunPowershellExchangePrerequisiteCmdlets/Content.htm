To learn how to install the necessary operating system components for Exchange using PowerShell cmdlets, proceed with the following steps...
  • Begin first by selecting the 'PowerShell' button at the bottom left corner of your screen.
  • Once PowerShell is open, type the following cmdlet and then press 'Enter':

Import-Module ServerManager

 
  • Then type the following cmdlet to install the necessary operating system components and press 'Enter':

Add-WindowsFeature NET-Framework,RSAT-ADDS,Web-Server,Web-Basic-Auth,Web-Windows-Auth,Web-Metabase,Web-Net-Ext,Web-Lgcy-Mgmt-Console,WAS-Process-Model,RSAT-Web-Server,Web-ISAPI-Ext,Web-Digest-Auth,Web-Dyn-Compression,NET-HTTP-Activation,RPC-Over-HTTP-Proxy,Desktop-Experience -Restart


  • Wait while PowerShell installs the necessary operating system components. When the installation completes, the server will automatically restart.
  • After the server restarts, select the 'PowerShell' button again.
  • Then type the following cmdlet and press 'Enter':

Set-Service NetTcpPortSharing -StartupType Automatic
