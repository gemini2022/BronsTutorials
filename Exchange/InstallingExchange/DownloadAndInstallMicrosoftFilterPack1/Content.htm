Along with installing exchange, you have to also install the Microsoft Office Filter Pack. The iFilters in the filter pack are important to Exchange as they are used by the MSSearch component to create the content indexes for mailbox databases. The indexes are used for searching by online clients (Outlook configured in cached Exchange mode uses Windows Desktop Search) and eDiscovery. They’re also used by the transport system if access is necessary to message content, such as when a transport rule needs to identify content as messages pass through the pipeline. To learn how to download and install the Microsoft Office Filter Pack, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then launch Internet Explorer by clicking on the 'Internet Explorer' icon.
  • Once Internet Explorer is open, type the following address in the address bar to get to the Microsoft Filter Pack download page: 'https://www.microsoft.com/en-us/download/details.aspx?id=20109'.

 
  • Once you get to the Microsoft Filter Pack download page, scroll down to the middle of the page.
  • And then click on the 'Download' button.
  • After clicking on the 'Download' button, select the option 'FilterPackx64.exe'.
  • And then click 'Next'.
  • Next, click 'Run' on the 'Open File' dialog box to open the 'Microsoft Filter Pack 1.0 Setup Wizard'.
  • Once the 'Microsoft Filter Pack 1.0 Setup Wizard' is open, click the 'Next >' button.
  • Then accept the terms in the license agreement by selecting the 'I accept the terms in the License Agreement' checkbox.
  • Click 'Next >'.
  • Then wait while the 'Microsoft Filter Pack 1.0' installs.
  • Once the 'Microsoft Filter Pack 1.0' installation is complete, click 'OK'.
