Use the 'Group Information' tab to change the Active Directory information for the dynamic distribution group. To learn about the Group Information tab, proceed with the following steps...
  • Use this box to view or change the name of the group. The name can't exceed 64 characters.
  • The recipient that's designated as the manager for this dynamic distribution group will be visible when users view the properties of this group in Outlook or Outlook Web App. If the delivery reports option on the Advanced tab is set to Send delivery reports to group manager, the manager will also receive delivery reports for the group. To choose the recipient you want to add as a manager of the group, begin first by selecting this checkbox.
  • Then click the 'Browse' button to open the 'Select Mailbox or Mail-Enabled user or Mail Contact' dialog box.
  • Once the 'Select Mailbox or Mail-Enabled user or Mail Contact' dialog box is open, select the recipient that you want to add as a manager of the group. In this case, we will choose the 'Administrator', which is already selected by default.
  • After a selection has been made, click 'OK'.
  • Once the 'OK' button is clicked on the 'Select Mailbox or Mail-Enabled user or Mail Contact' dialog box, the recipient you selected will be displayed here.
  • Use this box to view or modify administrative notes about the group. These notes are also visible in Outlook. When a user views the properties of the group in Outlook, the notes will be displayed on the General tab.
