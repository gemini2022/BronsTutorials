Use the 'E-Mail Addresses' tab to configure the e-mail addresses for the recipient. You can modify the existing addresses or create additional ones. Each recipient must have at least one primary STMP address that is internal to your Exchange organization and one external address. To learn about the E-Mail Addresses tab, proceed with the following steps...
  • This section displays all e-mail address configured for the dynamic distribution group.
  • To add a new e-mail address for this dynamic distribution group, begin by first clicking the 'Add' button to open the 'SMTP Address' dialog box.
  • Once the 'SMTP Address' dialog box is open, enter the new e-mail address here.
  • In this case, we will enter 'email@bronstutorials.local'.
  • Once the new e-mail address has been entered, click 'OK'.
  • After the 'OK' button on the 'SMTP Address' dialog box has been clicked, the new e-mail address will be displayed here.

  • Click this button to set your selected address as the "reply to" address. A recipient can have multiple e-mail addresses for a specific address type. This allows the recipient to receive messages that are addressed to any one of these e-mail addresses. However, a single address must be used for any messages that are sent by the recipient. If a recipient has multiple e-mail addresses, the primary address is used for any messages sent by the recipient. This button is available only when an address other than the primary address is selected. Primary addresses for each address type are displayed in bold type. If an e-mail address policy in your Exchange organization applies to this mailbox, the Set as Reply setting will be controlled by that policy. To change the primary address for a specific address type, you must clear the Automatically update e-mail addresses based on e-mail address policy check box.
  • Select this check box to have the recipient's e-mail addresses automatically updated based on changes made to e-mail address policies in your organization. This box is selected by default.
