Use the 'Conditions' tab to define any optional conditions to further restrict the recipients that are included in this dynamic distribution group. To learn how to do this, proceed with the following steps...
  • The first condition that can be set is 'Recipient is in a State or Province'. Select this check box if you want the dynamic distribution group to include only recipients from specific states or provinces. This information is contained on the Address and Phone tab in the recipient's properties.
  • The next condition is 'Recipient is in a Department'. Select this check box if you want the dynamic distribution group to include only recipients in specific departments. This information is contained on the Organization tab in the recipient's properties.
  • The next condition is 'Recipient is in a Company'. Select this check box if you want the dynamic distribution group to include only recipients in specific companies. This information is contained on the Organization tab in the recipient's properties.
  • And then finally, there are 15 custom attributes for each recipient. There is a separate condition for each custom attribute. If you want the dynamic distribution group to include only recipients that have a specific value set for a specific custom attribute, select the check box that corresponds to that custom attribute.
  • For each condition, click the underlined term to create your condition. By default, the underlined term for new conditions will read specified. After you edit the condition, the underlined term will change to the value that you specified.
