To learn how to send a message to a mailbox, proceed with the following steps...
  • Begin first by selecting a Dynamic Distribution Group. In this case we will select the new Dynamic Distribution Group we just created, 'DynamicDistributionGroup'.
  • Then select the option 'Send Mail' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • The operation couldn't be performed because the default mail client isn't properly installed. Click 'OK'.
