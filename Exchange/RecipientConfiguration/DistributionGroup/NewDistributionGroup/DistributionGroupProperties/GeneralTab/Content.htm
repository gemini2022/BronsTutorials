Use the 'General' tab to view or modify the display name, alias, and custom attributes for the distribution group. To learn how to do this, proceed with the following steps...
  • Use this unlabeled box at the top of the page to view or change the display name for the distribution group.
  • This read-only field displays the organizational unit (OU) that contains the distribution group.
  • This read-only field displays the last date and time a configuration change was made.
  • Use this box to view or change the alias of the distribution group. The alias cannot exceed 64 characters and must be unique in the forest.
  • And then finally, you can specify up to 15 custom attributes for the distribution group. To do this, click the 'Custom Attributes' button.
  • Then use the 15 corresponding boxes to specify custom attribute values.
  • Click 'OK' when finished.
