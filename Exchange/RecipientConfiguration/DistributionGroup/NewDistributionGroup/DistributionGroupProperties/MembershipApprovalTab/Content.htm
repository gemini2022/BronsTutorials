Use the 'Membership Approval' tab to configure how membership requests should be handled for this distribution group. To learn how to do this, proceed with the following steps...
  • Select this option to allow users to join this distribution group without the approval of the distribution group owners.
  • Select this option to specify that only distribution group owners can add members to this distribution group. Requests to join this distribution group will be rejected automatically.
  • Select this option to specify that users can request membership to this distribution group. Requests to join the distribution group must be approved by a distribution group owner before the user can join.
  • Select this option to allow users to leave this distribution group without the approval of the distribution group owners.
  • Select this option to specify that only distribution group owners can remove members from this distribution group. Requests to leave this distribution group will be rejected automatically.
