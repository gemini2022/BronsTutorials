Use the 'Group Information' tab to view or change the Active Directory information for the distribution group. To learn about the Group Information tab, proceed with the following steps...
  • Use this text box to view or change the name of the group. The name cannot exceed 64 characters.
  • Use this text box to type the name of the group that is compatible with legacy versions of Windows (prior to the release of Windows 2000 Server). The name of the group that is compatible with earlier versions of Windows can't exceed 20 characters. It can contain letters, numbers, and the following characters: ! # $ % ^ & - . _ { } | ~.
  • The recipient that is designated as the manager for this distribution group will be visible when users view the properties of this group in Outlook or Outlook Web App. If the delivery reports option on the Advanced tab is set to Send delivery reports to group manager, the manager will also receive delivery reports for the group. To add a manager for this distribution group, begin by clicking the 'Add' button to open the 'Select Mailbox or Mail-Enabled User' dialog box.
  • Once the 'Select Mailbox or Mail-Enabled User' dialog box is open, select a mailbox or mail-enabled user you want to designate as the manager for this distribution group. In this case, we will select 'Mail User'.
  • Once a selection has been made, click 'OK'.
  • After the 'OK' button on the 'Select Mailbox or Mail-Enabled User' dialog box has been clicked, the mailbox or mail-enabled user you selected will be displayed here.
  • Use this box to view or modify administrative notes about the distribution group. These notes are also visible from Outlook. When a user views the properties of the distribution group in Outlook, the notes will be displayed on the General tab.
