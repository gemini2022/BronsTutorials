Use the 'Mail Flow Settings' tab to configure message size or message delivery restrictions for the distribution group. To learn how to do this, proceed with the following steps...
  • With this first setting, 'Message Size Restrictions', we have the option of configuring message size restriction for this distribution group. To do this, begin by first selecting this setting.
  • Then click the 'Properties' button to open the 'Message Size Restrictions' dialog box.
  • Once the 'Message Size Restrictions' dialog box is open, we can set the maximum size of messages that can be received. To do this, begin by selecting the 'Maximum message size' checkbox.
  • Then enter the maximum size of a message that this distribution group can receive.
  • In this case, we will enter '1000'.
  • Click 'OK'.
  • Next, we have the option of configuring message delivery restrictions for this distribution group. To do this, begin by first selecting the setting 'Message Delivery Restrictions'.
  • Then click the 'Properties' button to open the 'Message Delivery Restrictions' dialog box.
  • Once the 'Message Delivery Restrictions' dialog box is open, the first thing we can do is select this option to specify that this distribution group can accept messages from all senders. This includes senders in both the Exchange organization and external senders. This option is selected by default.
  • Or select this option to specify that this distribution group can accept messages only from a specified set of senders in your Exchange organization.
  • After this option is selected, click the 'Add' button to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, a list of all recipients in the Active Directory forest will be displayed. From here, select the recipients you want this distribution group to only accept messages from. In this case we will select 'Distribution Group'.
  • After a selection is made, click 'OK'.
  • Once the 'OK' button on the 'Select Recipient' dialog box is clicked, the recipient you selected will be displayed here.
  • Select this check box to prevent anonymous users from sending messages to this distribution group. If you select this check box, messages from external users will be rejected.
  • Select this option to specify that this distribution group will not reject messages from any senders in the Exchange organization. This button is selected by default.
  • Or select this option to specify that this distribution group will reject messages from a specified set of senders in your Exchange organization.
  • Once this option is selected, click the 'Add' button to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, a list of all recipients in the Active Directory forest will be displayed. From here, select the recipients you want this recipient to reject messages from. In this case, we will choose the 'Administrator', which is already selected by default.
  • After the recipient is selected, click 'OK'.
  • Once the 'OK' button on the 'Select Recipient' dialog box is clicked, the recipient you selected will be displayed here.
  • Click 'OK'.
  • And then finally, we have the option of configuring message moderation for this distribution group. Message moderation is the proccess by which incoming messages are reviewed by one or more moderators before delivery. Moderators can then approve or reject the incoming message. To configure message moderation, begin by first selecting the setting 'Message Moderation'.
  • Then click the 'Properties' button to open the 'Message Moderation' dialog box.
  • Once the 'Message Moderation' dialog box is open, select this check box to require that all messages sent to this group are to be approved by a moderator.
  • Then we need to select the recipients we want to add as moderators for this distribution group. To do this, begin by first clicking the 'Add' button to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, select the recipient you want to use as the moderator. In this case, we will choose the 'Administrator', which is already selected by default.
  • After a selection has been made, click 'OK'.
  • Once the 'OK' button is clicked on the 'Select Recipient' dialog box, the recipient you selected will be displayed here.
  • Next, we can select the recipients who don't require message approval to send to the distribution group To do this, begin by first clicking the 'Add' button to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, select the recipients that don't require message approval to send to this distribution group. In this case, we will choose the 'Administrator', which is already selected by default.
  • After a selection has been made, click 'OK'.
  • Once the 'OK' button is clicked on the 'Select Recipient' dialog box, the recipient you selected will be displayed here.
  • Select this option to notify all senders when their messages aren't approved. This is the default setting. This option includes all people and groups in the address book.
  • When this option is selected, only people or groups in your organization are notified when a message they sent to the group isn't approved by a moderator.
  • And then finally, when this option is selected, notifications aren't sent to message senders whose messages aren't approved by the group moderators.
  • Click 'OK'.
