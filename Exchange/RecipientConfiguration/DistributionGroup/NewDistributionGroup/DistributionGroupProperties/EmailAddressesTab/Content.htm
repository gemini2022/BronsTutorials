Use the 'E-Mail Addresses' tab to configure the e-mail addresses for the distribution group. You can modify the existing addresses or create additional ones. Each recipient must have at least one primary STMP address that is internal to your Exchange organization and one external address. To learn about the E-Mail Addresses tab, proceed with the following steps...
  • This section displays all e-mail address configured for the distribution group.
  • To add a new e-mail address for this distribution group, begin by first clicking the 'Add' button to open the 'SMTP Address' dialog box.
  • Once the 'SMTP Address' dialog box is open, enter the new e-mail address here.
  • In this case, we will enter 'email@bronstutorials.local'.
  • Once the new e-mail address has been entered, click 'OK'.
  • After the 'OK' button on the 'SMTP Address' dialog box has been clicked, the new e-mail address will be displayed here.
  • Select this check box to have the recipient's e-mail addresses automatically updated based on changes made to e-mail address policies in your organization. This box is selected by default.
