Use the 'Members' tab to add members to or remove members from the distribution group. To learn how to do this, proceed with the following steps...
  • To add recipients to the distribution group, begin first by clicking the 'Add' button to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, select the recipients that you want to add to this distribution group. In this case, we will choose the 'Administrator', which is already selected by default.
  • After a selection has been made, click 'OK'.
  • Once the 'OK' button is clicked on the 'Select Recipient' dialog box, the recipient you selected will be displayed here.
