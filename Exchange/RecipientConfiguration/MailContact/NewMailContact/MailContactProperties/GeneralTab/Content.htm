Use the 'General' tab to modify the display name, alias, and custom attributes. To learn how to do this, proceed with the following steps...
  • Use this unlabeled box at the top of the page to view or change the display name.
  • This field displays the organizational unit (OU) that contains the user account.

  • This read-only field displays the last date and time that a configuration change was made to the recipient. Configuration changes made through any other method, such as the through the Exchange Management Shell or Active Directory Service Interfaces (ADSI) Edit, will also update this field.
  • Use this box to view or change the recipient's alias. The alias can't exceed 64 characters and must be unique in the forest. One of the reasons why the alias field must be unique is because it's used to generate the SMTP address in a default installation.
  • Use this dropdown to select a rich text format option.
  • The first option 'Never', specifies that messages sent to this recipient will be converted to plain text.
  • This next option 'Always', specifies that messages sent to this recipient will be in the MAPI rich text format (RTF).
  • And then finally, this last option 'Use Default Settings', specifies that messages sent to this recipient will be sent in either MAPI RTF or plain text, depending on the settings of the client computer from which the message is sent.
  • Select this check box to prevent the recipient from appearing in the global address list (GAL) and other address lists that are defined in your Exchange organization. After you select this check box, users in your Exchange organization can still send messages to the recipient by using the e-mail address.
  • And then finally, you can specify up to 15 custom attributes for this mail contact. To do this, click the 'Custom Attributes' button.
  • Then use the 15 corresponding boxes to specify custom attribute values.
  • Click 'OK' when finished.
