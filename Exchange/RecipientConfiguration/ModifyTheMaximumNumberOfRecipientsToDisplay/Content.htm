The Maximum Number of Recipients to Display dialog box is used to specify how many recipients you want to display in the result pane of the Exchange Management Console (EMC). By default, a maximum of 1,000 recipients is displayed. Increasing this value can be beneficial in large environments. However, increasing the value also increases the time it takes to display the results. Depending on the size of your organization, it may also have a performance impact on the domain controller to which you are connected. To learn how to set the maximum number of recipients to display, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then open the 'Exchange Management Console' by selecting the 'Exchange Management Console' icon.
  • Once the 'Exchange Management Console' is open, expand the node 'Microsoft Exchange On-Premise'.
  • Then select 'Recipient Configuration'.
  • After 'Recipient Configuration' is selected, select the option 'Modify the Maximum Number of Reipients to Display' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • In the 'Maximum number of recipients to display' box, type the number of recipients that you want to display in the result pane. The number must be between 1 and 100,000.
