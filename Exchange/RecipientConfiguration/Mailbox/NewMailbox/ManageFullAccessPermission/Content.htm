Use the Manage Full Access Permission wizard to grant Full Access permissions to users or groups for a selected mailbox. When you grant the Full Access permission to another user for a mailbox, that user becomes able to log on to the mailbox and access its entire contents. To learn how to Manage Full Access Permission for a mailbox, proceed with the following steps...
  • Begin first by selecting a mailbox. In this case we will select the new mailbox we just created, 'New Mailbox'.
  • Then select the option 'Manage Full Access Permission' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • First, you will notice that the only user that has full access permissions for this mailbox is the owner.
  • In this case, we want to grant another user full access permisstions for this mailbox. To do this, begin first by clicking 'Add' to open the 'Select User or Group' dialog box.
  • Once the 'Select User or Group' dialog box is open, we need to then select the users or groups to which we want to grant full access permissions for. In this case, we will select the Administrator.
  • Once a user or group is selected, click 'OK'.
  • After clicking 'OK' on the 'Select User or Group' dialog box, the user or group you selected will be displayed here.
  • Click 'Manage'.
  • On this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
