A move request is the process of moving a mailbox from one mailbox database to another. A local move request is a mailbox move that occurs within a single forest. To learn how to perform a Local Move Request on a mailbox, proceed with the following steps...
  • Begin first by selecting a mailbox. In this case we will select the new mailbox we just created, 'New Mailbox'.
  • Then select the option 'New Local Move Request' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • This box displays the mailboxes that you selected in the result pane. If you want to add or remove mailboxes, click Cancel, and then make the changes in the result pane.
  • Next, we need to select the mailbox database to which we want to move the mailbox to. To do this, begin first by clicking 'Browse' to open the 'Select Mailbox Database' dialog box.
  • Once the 'Select Mailbox Database' dialog box is open, a list of all the mailbox databases will be displayed. In this case, the first selected mailbox database is the database where our mailbox currently resides.
  • We want to move the mailbox to this database.
  • Once a mailbox database is selected, click 'OK'.
  • Then Click 'Next >' to continue.
  • Next, we need to specify how we want to manage corrupted messages. The first option, 'Skip the mailbox', specifies that mailboxes containing corrupted messages won't be moved.
  • And then with this option 'Skip the corrupted messages', no corrupted messages will be moved.
  • If you select this option, you need to set the 'Maximum number of messages to skip'. It is recommended selecting this option only if the move request failed in a previous attempt.
  • In this case, we will choose the option 'Skip the mailbox'.
  • Click 'Next >'.
  • On this screen, review the 'Configuration Summary' and verify that the information you entered is correct. If you are not satisfied with any of the information, you may go back and make any changes necessary.
  • If the information in the 'Configuration Summary' is correct, press 'New'.
  • And then finally, on this last screen, you have the option of reviewing the associated shell command that was performed by this wizard.
  • Press 'Finish' to complete the wizard.
