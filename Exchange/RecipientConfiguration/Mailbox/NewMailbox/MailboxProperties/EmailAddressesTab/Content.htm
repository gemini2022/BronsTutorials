Use the 'E-Mail Addresses' tab to configure the e-mail addresses for the recipient. You can modify the existing addresses or create additional ones. Each recipient must have at least one primary STMP address that is internal to your Exchange organization and one external address. To learn about the E-Mail Addresses tab, proceed with the following steps...
  • This address type is available only for user mailboxes. It's not available for mail users, mail contacts, distribution groups, or mail-enabled public folders. An EUM (Exchange Unified Messaging) address is used by Unified Messaging servers to locate UM-enabled users within an Exchange 2010 organization. EUM addresses contain the extension number and the UM dial plan for the UM-enabled user.
  • This is the default address type.
  • To add a new e-mail address for this recipient, click 'Add'.
  • Then enter the email address.
  • In this case, the email address we will enter will be 'email@bronstutorials.local'.
  • Once the email address is entered, click 'OK'.
  • After clicking 'OK' on the dialog box, the email that you entered will be displayed here.
  • Select this check box to have the recipient's e-mail addresses automatically updated based on changes made to e-mail address policies in your organization. This box is selected by default.
