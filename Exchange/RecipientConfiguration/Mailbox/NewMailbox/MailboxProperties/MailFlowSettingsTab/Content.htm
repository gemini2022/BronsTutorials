Use the Mail Flow Settings tab to configure delivery options and message size or message delivery restrictions for the mailbox. To learn how to do this, proceed with the following steps...
  • First, we have the option of configuring delivery options for this mailbox. To do this, begin by first selecting the setting 'Delivery Options'.
  • Then click the 'Properties' button to open the 'Delivery Options' dialog box.
  • Once the 'Delivery Options' dialog box is open, the first thing we can do is grant a recipient permissions to send e-mail messages on behalf of this selected mailbox. To do this, begin by clicking 'Add' to open the 'Select Mailbox or Mail-enabled User' dialog box.
  • Then select the recipient you want to grant permissions to. In this case, we will choose the 'Administrator', which is already selected by default.
  • Then click 'OK'.
  • After the 'OK' button is clicked on the 'Select Mailbox or Mail-enabled User' dialog box, the recipient that you selected will be displayed here.
  • Next, select this check box to allow all e-mail messages that are sent to this mailbox to be forwarded to a specified recipient.
  • Once this option is selected, click the 'Browse' button to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, select the recipient you want to forward all e-mail messages to. In this case we will select 'DistributionGroup'.
  • After the recipient is selected, click 'OK'.
  • Once the 'OK' button is clicked on the 'Select Recipient' dialog box, the recipient that you selected will be displayed here.
  • If you selected the 'Forward to' check box, you can then select this check box to specify that e-mail messages be delivered to both the mailbox and to the forwarding address.
  • Select this check box to limit the number of recipients to which this mailbox can send e-mail messages at one time.
  • Then enter how many recipients this mailbox can send e-mail messages to at one time.
  • In this case, we will enter '10'.
  • Click 'OK'.
  • Next, we have the option of configuring message size restrictions for this mailbox. To do this, begin by first selecting the setting 'Message Size Restrictions'.
  • Then click the 'Properties' button to open the 'Message Size Restrictions' dialog box.
  • Once the 'Message Size Restrictions' dialog box is open, the first thing we can do is set the maximum size of messages that can be sent by this recipient. To do this, begin by selecting the 'Maximum message size' checkbox.
  • Then enter the maximum size of a message that this recipient can send.
  • In this case, we will enter '5000'.
  • And then we can set the maximum size of messages that this recipient can receive. To do this, begin by selecting this 'Maximum message size' checkbox.
  • Then enter the maximum size of a message that this recipient can receive.
  • In this case, we will enter '5000'.
  • Click 'OK'.
  • And then finally, we have the option of configuring message delivery restrictions for this mailbox. To do this, begin by first selecting the setting 'Message Delivery Restrictions'.
  • Then click the 'Properties' button to open the 'Message Delivery Restrictions' dialog box.
  • Once the 'Message Delivery Restrictions' dialog box is open, the first thing we can do is select this option to specify that this recipient can accept messages from all senders. This includes senders in both the Exchange organization and external senders. This option is selected by default.
  • Select this option to specify that this recipient can accept messages only from a specified set of senders in your Exchange organization.
  • After this option is selected, click the 'Add' button to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, a list of all recipients in the Active Directory forest will be displayed. From here, select the recipients you want this recipient to only accept messages from. In this case we will select 'Distribution Group'.
  • After the recipient is selected, click 'OK'.
  • Once the 'OK' button on the 'Select Recipient' dialog box is clicked, the recipient you selected will be displayed here.
  • Select this check box to prevent anonymous users from sending messages to the recipient. If you select this check box, messages from external users will be rejected.
  • Select this option to specify that the recipient will not reject messages from any senders in the Exchange organization. This button is selected by default.
  • Or select this option to specify that the recipient will reject messages from a specified set of senders in your Exchange organization.
  • Once this option is selected, click the 'Add' button to open the 'Select Recipient' dialog box.
  • Once the 'Select Recipient' dialog box is open, a list of all recipients in the Active Directory forest will be displayed. From here, select the recipients you want this recipient to reject messages from. In this case, we will choose the 'Administrator', which is already selected by default.
  • After the recipient is selected, click 'OK'.
  • Once the 'OK' button on the 'Select Recipient' dialog box is clicked, the recipient you selected will be displayed here.
  • Click 'OK'.
