Use the Find dialog box (a search feature) to help search for specific recipients by name. To learn how to do this, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then open the 'Exchange Management Console' by selecting the 'Exchange Management Console' icon.
  • Once the 'Exchange Management Console' is open, expand the node 'Microsoft Exchange On-Premise'.
  • Then select 'Recipient Configuration'.
  • After 'Recipient Configuration' is selected, select the option 'Find' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • Then use this text box to type the search string to use to find recipients.
  • In this case, we will type 'administrator'.
  • Then, to start the search, click 'Find Now'. The search returns all recipients whose first name, last name, display name, user principal name (UPN), alias, or e-mail address begins with the specified search string. For example, if you search for the string ter, the search results include the recipient Terry Adams, but not Adam Carter.
  • When a search completes, the recipient objects that match your criteria are displayed in the result pane.
  • If you double-click a recipient in the result pane.
  • Then the property page for that recipient is displayed.
  • Click 'OK'.
  • Or, if you right-click a recipient.
  • The shortcut menu displays the actions you can perform for that recipient type.
