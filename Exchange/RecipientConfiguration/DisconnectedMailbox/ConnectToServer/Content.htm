To learn how to connect to a server, proceed with the following steps...
  • Begin by selecting the option 'Connect to Server' found in the Actions pane located at the far right side of the Exchange Management Console window.
  • First, click 'Browse' to open the 'Select Exchange Server' dialog box.
  • Once the 'Select Exchange Server' dialog box is open, select the Exchange server you want to connect to. In this case there is only one Exchange server, which is already selected by default.
  • Click 'OK'.
  • Next, you have the option of setting the selected Exchange server as the default server.
  • Finally, click 'Connect' to connect to the selected Exchange server.
