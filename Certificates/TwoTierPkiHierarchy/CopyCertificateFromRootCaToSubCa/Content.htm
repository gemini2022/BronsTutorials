In this step, we're going to copy the certificate from the offline Standalone Root CA server to the Enterprise Subordinate CA server so that the certificate can later be installed in the Enterprise Subordinate CA store. To learn how to do this, proceed with the following steps...
  • If not already, log into the offline Standalone Root CA server.
  • Then go to the desktop where the certificate was saved in the previous step, and right-click on the certificate.
  • And select 'Copy'.
  • Now, because this server is offline, the only way of getting the certificate to the Enterprise Subordinate CA server is by some form of removable media. Insert the removable media of your choice into the Standalone Root CA server.
  • Once the removable media of your choice has been inserted, right-click in the window...
  • And select 'Paste'.
  • After the certificate has been pasted, safely remove the removable media from the server, and make your way to the other server where the Enterprise Subordinate CA has been installed.
  • Log into the Enterprise Subordinate CA server.
  • Now insert the removable media that has a copy of the certificate into the Enterprise Subordinate CA server.
  • And right-click on the certificate.
  • And select 'Copy'.
  • Now go to the desktop of the Enterprise Subordinate CA server and right-click.
  • And select 'Paste'.
  • The certificate has now been coppied to the Enterprise Subordinate CA server.
