When the DHCP user classes are created and configured, the DHCP client computers must also be configured accordingly so that they can receive the settings that have been defined in the manually created DHCP user classes. For example, if the Accounts section in an organization is required to be configured to use a separate DNS server and the default gateway to communicate with the Internet, the DHCP administrators can create a new DHCP user class for the Accounts department, and can configure all the computers in the department to receive the settings as defined in the user class. This way, every time the DHCP client computers from the Accounts department request an IP address from the DHCP server, the settings that the DHCP administrators have configured for the Accounts DHCP user class would be assigned to the DHCP client computers in the Accounts section. Working with the DHCP user classes requires a three-step configuration where in the first step you are required to define a new DHCP user class, in the second step you configure the user class options, and the thrid step you configure the client computer to receive the DHCP options from the user class. To learn how to create a user class, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, expand your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Then select 'IPv4'.
  • Once 'IPv4' has been selected, right-click it...
  • And then select 'Define User Classes...'.
  • Once the 'DHCP User Classes' dialogue box opens, press the 'Add...' button.
  • Then begin by entering in a name for the new class. In this case, since we're creating a user class for the Accounts department, we'll name the new class 'Accounts'.
  • You can also add a description if you wish. This field is optional, but for this example, we'll enter 'DHCP user class for Accounts Department'.
  • And then here under the 'ASCII' column, enter in the class ID. This ID will be used for the client to receive IP information from this class. In this case we'll enter the ID of 'Accounts'.
  • Press 'OK'.
  • Then press 'Close'.
  • Now that the user class has been defined, we next have to apply options of the user class to a scope. We can apply these options to a single scope or all IPv4 scopes using the 'Server Options'. In this case, we will apply these options to a single scope. To do this, we'll first expand the scope.
  • Then select 'Scope Options'.
  • Once 'Scope Options' is selected, right-click it.
  • And select 'Configure Options...'.
  • Once the 'Scope Options' has been opened, select the 'Advanced' tab.
  • We now have to select the user class we just created. To do this, select the 'User class' dropdown.
  • And then select the newly created user class, in this case it's 'Accounts'.
  • From here you can configure any options needed for this class. For this example we'll select the option 'Router'.
  • Then enter in the option details. Here this option requires the IP address for the router.
  • Once the IP address has been entered, press the 'Add' button.
  • You can configure as many options as needed, when finished, press the 'OK' button.
  • Now, in order for the client computers in the Accounts department to receive the user class options we just created, we have to configure these client computers. To do this, first go to the client computer you want to configure and select 'Start' or the 'Start Orb' at the bottom left corner of your screen.
  • And type 'cmd' in the search field.
  • Then select the 'cmd' icon to open the command prompt.
  • Once the command prompt has been opened, type in 'ipconfig'.
  • Followed by '/setclassid'.
  • Then type in the name of the network adapter of the client computer in quotes. In this case, the name of the network adapter for this client computer is '"Local Area Connection"'.
  • And then finally, type the name of the class ID we created earlier. In this case the class ID is 'Accounts'.

  • Press 'Enter'.
