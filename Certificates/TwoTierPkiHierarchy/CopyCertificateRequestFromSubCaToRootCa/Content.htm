In this step, we need to copy both the crl and crt file from the Standalone Root CA to the Enterprise Subordinate CA. To learn how to do this, proceed with the following steps...
  • Begin first by logging into the server where the Standalone Root CA was installed.
  • Begin first by logging into the server where the Standalone Root CA was installed.
  • Begin first by logging into the server where the Standalone Root CA was installed.
  • Once logged in, we'll need to get to the 'CertEnroll' folder where the crl and crt file resides. To do this, go to 'C:\Windows\System32\certsrv\CertEnroll'.
  • Once there, select both files...
  • Then right-click
  • And select 'Copy'.
  • Now, because this server is offline, the only way of getting the two coppied files to the Enterprise Subordinate CA server is by some form of removable media. Insert the removable media of your choice into the Standalone Root CA server.
  • Once the removable media of your choice has been inserted, right-click in the window...
  • And select 'Paste'.
  • After the two files have been pasted, safely remove the removable media from the server, and make your way to the other server where the Enterprise Subordinate CA has been installed.
  • Log into the Enterprise Subordinate CA server.
  • Now insert the removable media that has a copy of the crl and crt file into the Enterprise Subordinate CA server.
  • And select both files.
  • Once both files have been selected, right-click
  • And select 'Copy'.
