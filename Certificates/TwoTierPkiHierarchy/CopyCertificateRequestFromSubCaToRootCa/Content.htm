In this step, we need to copy the certificate request from the Enterprise Subordinate CA to the Offline Standalone Root CA. To learn how to do this, proceed with the following steps...
  • If you recall, when installing the Enterprise Subordinate CA, we chose the option to save the certificate request to the 'C' drive on the Enterprise Subordinate CA server. This request needs to be accessible to the Offline Standalone Root CA. However, since the Standalone Root CA is offline, we'll need to manually move or copy the file to the Root CA server by some form of removable media.
  • To do this, if not already, log into Enterprise Subordinate CA server.
  • Then make your way to the 'C' drive on the Enterprise Subordinate CA server.
  • And here you should see the certificate request (provided you did save it to this location).
  • Next, right-click on the file.
  • And select 'Copy'.
  • Now again, because the Root CA server is offline, the only way of getting the certificate request to the Root CA server is by some form of removable media. Insert the removable media of your choice into the Enterprise Subordinate CA server.
  • Once the removable media of your choice has been inserted, right-click in the window...
  • And select 'Paste'.
  • After the certificat request has been pasted, safely remove the removable media from the server, and make your way to the other server where the Standalone Root CA has been installed.
  • Log into the offline Standalone Root CA server.
  • Now insert the removable media that has a copy of the certificate request file into the Standalone Root CA server.
  • And right-click on the certificate request.
  • And select 'Copy'.
  • The certificate request file can be put anywhere. In this case we will save a copy of the file on the desktop. To do this, we'll right-click on the desktop.
  • And select 'Paste'.
  • The certificate request file has now been coppied to the Standalone Root CA server.
