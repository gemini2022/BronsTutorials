To set up certificate autoenrollment, you must first create a new certificate template by duplicating an existing template and then giving the appropriate users, machines, or groups the Autoenroll permission. To learn how to do this, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, expand the root node for the CA, which in this case is 'EnterpriseCA'.
  • Then right-click 'Certificate Templates'.
  • And select 'Manage'.
  • As you can see, in the window to the right, a list of preconfigured certificate templates. The next step is to choose the template you wish to duplicate. In this case, the certificate template we will duplicate is the 'User' template. To duplicate this template, right-click it.
  • And select 'Duplicate Template'.
  • Next, select the template version that is appropriate for your deployment. In this case, we will select 'Windows Server 2008 Enterprise'. This option will give us additional options that can be configured.
  • Press 'OK'.
  • Once you press 'OK' on the previous dialogue box, the properties of the new template will appear. On the 'General' tab, you first have the option to rename the template display name, or keep the default setting.
  • In this case, we will rename the template 'User v2'.
  • More options can be configured, but for this demonstration we will leave everything else with the default settings.

  • Select the 'Security' tab.
  • How a client obtains a certificate is primarily controlled by the security properties of the certificate template. In this case, the client must be granted Read, Enroll, and Autoenroll permissions.
  • Ideally you would most likely assign Autoenroll permissions to a group, but for this example we will assign Autoenroll permissions to a user. To do this, first press the 'Add...' button.
  • Then enter in the user you want to give Autoenroll permissions to. In this case we will enter 'john p. doe'.

  • Once finished, press 'OK'.
  • Now add both Enroll and Autoenroll to the selected user.
  • That's it! Press 'OK' when finished.
