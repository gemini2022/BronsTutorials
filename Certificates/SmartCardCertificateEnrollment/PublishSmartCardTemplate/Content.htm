After creating a new certificate template, the next step is to deploy the certificate template so that a certification authority can issue that certificate.  To learn how to do this, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, expand the root node for the CA, which in this case is 'EnterpriseCA'.
  • Then right-click 'Certificate Templates'.
  • And select 'New'.
  • And then 'Certificate Template to Issue'.
  • After selecting the 'Certificate Template to Issue' option, a list of the preconfigured certificate templates will appear. Among this list, is the certificate template we created earlier 'Smart Card User Template'. In this case, to get to it, we need to scroll down to the bottom of the list.
  • Once the new template is located, select it.
  • Then press 'OK'.
  • Now, select 'Certificate Templates'.
  • And you can see our new certificate template 'Smart Card User Template' is in the list and ready to use.
