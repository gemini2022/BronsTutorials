In this step we will be enrolling for a smart card certificate on behalf of another user. To learn how to do this, proceed with the following steps...
  • Begin first by logging into a system that has a smart card reader with a user that has an enrollment agent certificate. Then select 'Start' or the 'Start Orb' at the bottom left corner of your screen.
  • Then type 'certmgr.msc' in the search field.
  • Now click on 'certmgr'.
  • Once in certmgr, right-click on the 'Personal' folder.
  • And, select 'All Tasks'.
  • And then 'Advanced Operations'.
  • Then 'Enroll On Behalf Of...'. This will launch the 'Certificate Enrollment' wizard.
  • Once the 'Certificate Enrollment' wizard is open, press 'Next' to begin the certificate enrollment process..
  • On this screen, press 'Next'.
  • On this screen, we need to first obtain the enrollment agent certificate. To do this, press the 'Browse..' button.
  • Then press 'OK' to obtain the certificate.
  • Now that you have the certificate, you can now enroll for a smart card on behalf of other users.

  • Press 'Next'.
  • On this screen, select the smart card template we created in an earlier step. This is the certificate we'll be enrolling on behalf of another user.
  • Then press 'Next'.
  • On this screen, we need to enter in the user we are going to enroll the smart card certificate on behalf of. To do this, first press 'Browse...'.
  • The user we are looking for is an account on the network. Before we can select this user, we first need to change the scope from the local computer to the domain. To do this, press the 'Locations...' button.
  • And then select the domain, which in this case is 'bronstutorials.local'.
  • Once the domain is selected, press 'OK'.
  • Now we can enter in the user. In this case, the user we will be enrolling the smart card certificate on behalf of is 'sally smith'.

  • Press 'OK'.
  • Now press 'Enroll'.
  • Wait while the request is being processed.
  • You will next be prompt to enter in a pin number for the smart card. Once this is complete, you may enroll on behalf of another user by pressing 'Next user', or press 'Close' to quit.
