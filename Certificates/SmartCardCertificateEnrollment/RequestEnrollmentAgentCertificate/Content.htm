In this step, we need to give the users that were designated as enrollment agents a certificate that will allow them to enroll for a smart card certificate on behalf of other users. To learn how to do this, proceed with the following steps...
  • Begin first by logging into a user that was designated as an enrollment agent. Then select 'Start' or the 'Start Orb' at the bottom left corner of your screen.
  • Then type 'certmgr.msc' in the search field.
  • Now click on 'certmgr'.
  • Once in certmgr, right-click on the 'Personal' folder.
  • And, select 'All Tasks'.
  • Then, select 'Request New Certificate...'. This will launch the 'Certificate Enrollment' wizard.
  • Once the 'Certificate Enrollment' wizard is open, press 'Next' to begin the certificate enrollment process.
  • On this screen, press 'Next'.
  • On this next screen, we need to select the type of certificate we want to enroll. In this case, we will select the 'Enrollment Agent' certificate. Obtaining this certificate will allow this user to enroll for a smart card certificate on behalf of other users.
  • Then press 'Enroll'.
  • Wait while the request is being processed.
  • Once the enrollment process has succeeded, press 'Finish'.
