Certificate Services publishes periodic certificate revocation lists (CRLs). However, you can also use the Certification Authority console to manually publish a new CRL at any time. For example, you might want to publish a new CRL immediately after revoking certificates. To learn how to publish a CRL, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, expand the root node for the CA, which in this case is 'RootCA'.
  • Then right-click 'Revoked Certificates'.
  • Select 'All Tasks'.
  • And then select 'Publish'.
  • Once 'Publish' is selected, you will then be presented with two options. The first option is 'New CRL'. This is the default selection, which will issue a complete CRL.
  • The second option 'Delta CRL only', issues an abbreviated version of the CRL, which contains only the updates to the CRL that have been made since the last time it was published.

  • In this case, we will publish the CRL using the 'New CRL' option. Press 'OK'.
