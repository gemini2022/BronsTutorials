To learn about the 'CRL Publishing Parameters' tab, proceed with the following steps...
  • The CRL Publishing Parameters tab has two sections. The first section relates to the CRLs.
  • In the CRL section, the 'CRL publication interval' field allows us to set when the CRL will automatically publish. In this example, we'll change the CRL publication interval from 1 week to 20 years. To do this, we will first set the first input field to '20'.
  • Then select the dropdown.
  • And select 'Years'.
  • The CRL will now publish every 20 years.
  • The second section of the 'CRL Publishing Parameters' tab relates to the delta CRLs.
  • Before we can edit this section, we first need to enable it.
  • In the delta CRL section, the 'Publication interval' field allows us to set when the delta CRLs will automatically publish. In this example, we want to change the delta CRL publication interval from 1 day to 2 weeks. To do this, we will first set the first input field to '2'.
  • Then select the dropdown.
  • And select 'Weeks'.
  • The delta CRL will now publish every 2 weeks.
