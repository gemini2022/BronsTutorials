To learn about the 'View CRLs' tab, proceed with the following steps...
  • The upper section of the tab displays overview information about the CRL.
  • We can get more detailed information about the CRL by clicking on the 'View CRL' button.
  • And then within the detailed list of information, we can select a field.
  • And get detailed information on that field.
  • If we then select the 'Revocation List' tab.
  • We can then veiw a list of the revoked certificates. If we then select a revoked certificate.
  • We can get detailed information about that revoked certificate.

  • Press 'OK'.
  • Now, the lower section of the 'View CRLs' tab displays overview information about delta CRLs.
  • We can get more detailed information about the delta CRLs by clicking on the 'View Delta CRL' button.
  • And then within the detailed list of information, we can select a field.
  • And get detailed information on that field.
  • If we then select the 'Revocation List' tab.
  • We can then veiw a list of the revoked certificates. In this case, we have none.

  • Press 'OK'.
