To learn how to duplicate a certificate, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, expand the root node for the CA, which in this case is 'EnterpriseCA'.
  • Then right-click 'Certificate Templates'.
  • And select 'Manage'.
  • As you can see, in the window to the right, a list of preconfigured certificate templates. The next step is to choose the template you wish to duplicate. In this case, the certificate template we will duplicate is the 'User' template. To duplicate this template, right-click it.
  • And select 'Duplicate Template'.
  • And select 'Duplicate Template'.
  • And select 'Duplicate Template'.
  • And select 'Duplicate Template'.
  • And select 'Duplicate Template'.
