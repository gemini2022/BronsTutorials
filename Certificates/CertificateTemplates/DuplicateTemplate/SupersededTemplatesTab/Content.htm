The Superseded Templates tab, allows you to supersede existing certificate templates with a modified version 2 certificate template. This tab allows you to redeploy certificates and ensure that the updated certificates replace previous versions of the certificate or other certificate templates that were used for the same purposes.
  • Begin first by pressing the 'Add...' button.
  • In this case, the template we want to supersede is located near the bottom of the list. So we will first scroll down.
  • And select the template we want to supersede, which in this case is the 'User' template.
  • Once the template has been selected, press 'OK'.
  • Now the 'User' template will be superseded with this template.
