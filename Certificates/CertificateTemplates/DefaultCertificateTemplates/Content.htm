When an Enterprise CA is installed, a set of default certificate templates is assigned to the CA so that the CA is immediately able to issue certificates for those templates.  To learn about these default certificate templates, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, expand the root node for the CA, which in this case is 'EnterpriseCA'.
  • Then select 'Certificate Templates'.
  • And in the window to the right will be a list of all the default certificate templates assigned and ready for enrollment.
