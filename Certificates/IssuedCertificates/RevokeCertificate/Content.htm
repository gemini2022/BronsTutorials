To learn how to revoke a certificate, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, expand the root node for the CA, which in this case is 'RootCA'.
  • Then select 'Issued Certificates'.
  • Once 'Issued Certificates' has been selected, look at the window to the right. There you will see a list of all the certificates that have been issued. In this case, there is only one certificate that has been issued.
  • To revoke this certificate, right-click it.
  • Then select 'All Tasks'.
  • And then 'Revoke Certificate'.
  • First, it's a good idea to specify as to why you are revoking the certificate. You can do this by selecting the dropdown.
  • And choosing one of the reasons provided in the dropdown list.
  • In this case, we'll select the reason 'Certificate Hold'. With this reason, it will allow this certificate to be unrevoked. Selecting any other reason in the list will not provide this certificate the option to be unrevoked.
  • You can choose to revoke this certificate at any time. You can choose to revoke this certificate now or revoke this certificate at a later date and time if you wish. To do this, select the 'Date and Time' dropdown.
  • Once you select the 'Date and Time' dropdown, a calendar will appear displaying the current date.
  • In this case, we want to change the date that this certificate will get revoked. To do this, press this button to change the month.
  • And then select the day of the month. In this case we'll select the '22nd'.
  • We can also change the time this certificate will get revoked. Start by selecting the hour.
  • And change the hour by advancing the value up or down.
  • Then select the minutes.
  • And change the minutes by advancing the value up or down.
  • And then finally we can set the time to 'AM' or 'PM'. Just select select it.

  • And change the value.
  • Press 'Yes' to have the certificate revoked at the specified date and time.
