In this step, we're going to issue the certificate request. To learn how to issue a certificate request, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, expand the root node for the CA, which in this case is 'RootCA'.
  • Then select 'Pending Requests'.
  • Once 'Pending Requests' has been selected, look at the window to the right. There you will see the certificate request we submitted in the previous step. The certificate request resides in the 'Pending Request' folder because it is waiting to be manually issued by an administrator.
  • To issue the certificate, right-click on it...
  • Then select 'All Tasks'.
  • And then finally, select 'Issue'.
