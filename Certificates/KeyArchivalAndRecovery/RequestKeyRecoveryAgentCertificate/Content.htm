Before a key recovery agent can use a key recovery certificate, the key recovery agent must enroll for the key recovery certificate and be registered as the recovery agent for the certification authority. To learn how to do this, proceed with the following steps...
  • Begin first by logging into a user that was designated as the 'Key Recovery Agent'. Then select 'Start' or the 'Start Orb' at the bottom left corner of your screen.
  • Then type 'certmgr.msc' in the search field.
  • Now click on 'certmgr'.
  • Once in certmgr, right-click on the 'Personal' folder.
  • And, select 'All Tasks'.
  • Then, select 'Request New Certificate...'. This will launch the 'Certificate Enrollment' wizard.
  • Once the 'Certificate Enrollment' wizard is open, press 'Next' to begin the certificate enrollment process.
  • On this screen, press 'Next'.
  • On this next screen, we need to select the type of certificate we want to enroll. In this case, we will select the 'Key Recovery Agent' certificate.
  • Then press 'Enroll'.
  • Wait while the request is being processed.
  • Once the enrollment process has succeeded, press 'Finish'.
  • Now, to confirm that the 'Key Recovery Agent' certificate has been issued, expand the 'Personal' store.
  • Then select the'Certificates' folder.
  • Once the 'Certificates' folder has been selected, you'll see in the window to the right, that a certificate has been issued.
  • If we now scroll to the far right.
  • We can see that the certificate that was issued was the 'Key Recovery Agent' certificate. Bob Smith is now authorized to be the Key Recovery Agent.
