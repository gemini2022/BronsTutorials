In order for smart card certificates to be issued, a user or a group needs to obtain an enrollment agent certificate. In this next step, we need to first configure an 'Enrollment Agent' template so that an enrollment agent can be indentified. To learn how to do this, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, expand the root node for the CA, which in this case is 'EnterpirseCA'.
  • Then right-click 'Certificate Templates'.
  • And select 'Manage'.
  • In this next step, instead of duplicating a template, we can just edit an existing one. The template we need to edit is the 'Enrollment Agent' template. To do this, right-click it.
  • And select 'Properties'.
  • Once in the properties of the 'Enrollment Agent' template, select the 'Security' tab.
  • Now, we need to make sure that the 'Help Desk' group has read and enroll permissions assigned on this template as well. But again, this group isn't in the list, so we need to add it. To do this, press the 'Add...' button.
  • Then type in the group, which again in this case is 'help desk'.

  • Press 'OK'.
  • Like before, the group already has 'Read', but needs 'Enroll'. To assign the group 'Enroll' permissions, select the 'Enroll' option under the 'Allow' column.
  • Then press 'OK'.
