This procedure can be completed only by an individual that has been issued a key recovery certificate To learn how to do this, proceed with the following steps...
  • Begin first by logging into a user that was designated as the 'Key Recovery Agent'. Then select 'Start' or the 'Start Orb' at the bottom left corner of your screen.
  • Then type 'cmd' in the search field.
  • Now click on 'cmd'.
  • Once in the command prompt, run the command 'certutil'.
  • Followed by '-recoverkey'.
  • Next, we have to specify the path and name of the file that contains the encrypted copy of the archived key we created in the previous step using the Certutil.exe –getkey command. In this case, we saved the file in a shared location on the network, '\\dc1\certfiles'.
  • And then, we named the file '\JaneDoeArchivedKey'.
  • And then finally, we need to provide a name of the file that the decrypted private key will be saved in. In this case, we will name this file 'JaneDoeRecoveredKey.pfx'.

  • Press 'Enter'.
  • The copy of the private key stored in the .pfx file is clear text, therefore this file must be protected by a password. Enter in a password and press 'Enter'.
  • Then confirm the password.

  • Press 'Enter'.
  • The file has now been created and saved to 'C:\Users\bsmith'.
