In this next step, now that the certificate request has been transferred from the Enterprise Subordinate CA server to the offline Standalone Root CA server, we now need to officially submit the certificate request. To learn how to submit the certificate request, proceed with the following steps...
  • If not already, log into the offline Standalone Root CA server.
  • And then select 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, right-click the root node for the CA, which in this case is 'RootCA'.
  • Then select 'All Tasks'.
  • And then select 'Submit new request...'.
  • If you recall, we placed a copy of the certificate request file on the desktop. We can get there from here by simply selecting 'Desktop'.
  • Once on the desktop, locate the certificate request and select it.
  • Then press 'Open'. Pressing the 'Open' button will submit the request to the Root CA.
