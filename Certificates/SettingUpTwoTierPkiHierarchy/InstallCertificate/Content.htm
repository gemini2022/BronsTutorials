In this step, we're going to install the certificate into the Enterprise Subordinate CA. To learn how to do this, proceed with the following steps...
  • If not already, log into Enterprise Subordinate CA server.
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, right-click on the root node for the CA, which in this case is 'IssuingCA'.
  • Then select 'All Tasks'.
  • And then 'Install CA Certificate...'.
  • If you recall, we placed a copy of the certificate on the desktop. We can get there from here by simply selecting 'Desktop'.
  • Once on the desktop, locate the certificate and select it.
  • Then press 'Open'.
  • Once the 'Open' button has been pressed, a dialogue box will appear asking if you trust the certificate you are about to install. Press 'OK' to install the certificate.
