Typically the Security tab contains different levels of Security setting for a file or folder. In this case, the Security tab sets security settings for the CA. To learn about the Security tab, proceed with the following steps...
  • If we take a look at the first group, 'Authenticated Users'...
  • By default, 'Authenticated Users' have only permission to request certificates.
  • If we select 'Domain Admins'.
  • We can see that by default, 'Domain Admins' have the permissions to issue and manage certificates and they can manage the CA.
  • If we select 'Enterprise Admins'.
  • We can see that by default, 'Enterprise Admins' also have the permissions to issue and manage certificates and manage the CA.
  • And then finally, 'Administrators'.
  • Also have the permissions to issue and manage certificates and manage the CA.
