To learn how to restore the CA, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, right-click the root node for the CA, which in this case is 'RootCA'.
  • First, select 'All Tasks'.
  • And then select 'Restore CA...'.
  • Before the CA can be restored, Active Directory Certificate Services needs to stop running. A dialogue box will appear asking if you want to stop Active Directory Certificate Services now.

  • Press 'OK'.
  • After Active Directory Certificate Services has stopped running, the 'Certificate Authority Restore Wizard' will appear.

  • Press 'Next >' to begin the restore process.
  • On this next screen, we need to select what part of the CA we want to restore. In this case we will restore both the 'Private key and CA certificate'.
  • And the 'Certificate database and certificate database log'.
  • Next, we need to navigate to the location where the CA backup file is located. To do this, first select 'Browse...'.
  • Then select the folder where the CA backup file is located. In this case, a copy of the backup file has been saved on the desktop. To get there, we first need to scroll up.
  • Then locate the desktop and select it.
  • Press 'OK'.
  • Now that the backup file has been located, press 'Next >'.
  • On this screen, we need to provide the password for the backup file. Only those who know this password will be able to restore the CA. To begin, enter the password.

  • Then press 'Next >'.
  • And then finally, press 'Finish' to restore the backup.
  • Once the restore operation has completed, a dialogue button will appear asking if you want to start Active Directory Services back up.

  • Press 'Yes'.
