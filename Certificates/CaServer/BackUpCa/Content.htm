To learn how to back up the CA, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, right-click the root node for the CA, which in this case is 'RootCA'.
  • First, select 'All Tasks'.
  • And then select 'Back up CA...'. This will open up the 'Certification Authority Backup Wizard'.
  • Once the 'Certification Authority Backup Wizard' has opened, press 'Next >'.
  • On this next screen, we need to select what part of the CA we want to back up. In this case we will back up both the 'Private key and CA certificate'.
  • And the 'Certificate database and certificate database log'.
  • Next, we need to define where the CA backup file will be located. To do this, first select 'Browse...'.
  • Then select the folder where you want to store the CA backup. For this example, we will save a copy of the backup on the desktop. To do this, we first need to scroll up.
  • Then locate the desktop and select it.
  • Press 'OK'.
  • Now that the destination for the backup file has been defined, press 'Next >'.
  • On this screen, we need to create a password for the backup file. Only those who will know this password will be able to restore the CA with this file. To begin, first enter a password.
  • Then confirm the password.

  • Press 'Next >'.
  • And then finally, press 'Finish' to create the backup.
