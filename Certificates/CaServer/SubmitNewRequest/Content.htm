To learn how to submit a certificate request, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Certification Authority'...
  • Once in Certification Authority, right-click the root node for the CA, which in this case is 'RootCA'.
  • First, select 'All Tasks'.
  • And then select 'Submit new request...'.
  • For this example, a copy of a certificate request was saved on the desktop. We can get there from here by simply selecting 'Desktop'.
  • Once on the desktop, locate the certificate request and select it.
  • Then press 'Open'. Pressing the 'Open' button will submit the request to the CA.
