To learn how to view details of the remote access clients, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'RRASSERVER (local)'.
  • Then select 'Remote Access Clients'.
  • Once you select 'Remote Access Clients', in the window to the right side of the Routing and Remote Access screen, you will see a list of details that pertain to all the clients that are currently connected. In this case, there is only one client connected at this time.
  • If we right-click on a remote access client.
  • There is then a couple of options worth taking a look at. One of them is 'Status...'. Selecting this option will allow us to view status details of that remote access client.
  • Or with this option, we have the ability to disconnect that remote access client if there is ever a need to.
  • If we go back and select the option 'Status...'.
  • You can see detailed information on the status of that remote access client.

  • Press 'Close'.
