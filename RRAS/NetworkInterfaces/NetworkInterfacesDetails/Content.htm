To learn how to view details of your network interface connections, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'RRASSERVER (local)'.
  • Then select 'Network Interfaces'.
  • Once you select 'Network Interfaces', in the window to the right side of the Routing and Remote Access screen, you will see a list of details that pertain to all the interface connections that are currently configured.
