To learn how to view details of the remote access ports, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'RRASSERVER (local)'.
  • Then select 'Ports'.
  • Once you select the 'Ports' node, you will see in the window to the right side of the Routing and Remote Access screen, a layout of details that pertain to all the ports that are available and which of these ports are currently being used. These ports represent the protocols that will be used to make a Routing and Remote Access connection.
