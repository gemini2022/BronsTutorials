To learn about the SSTP port, proceed with the following steps...
  • With the SSTP port you can select this option to enable incoming remote access connections. Deselect this option to prevent incoming connections for this port.
  • Here you can specify the IP address assigned to the Routing and Remote Access interface on the server through which the incoming connections arrive.
  • And then finally, here you can change the number of ports available for this port type.
