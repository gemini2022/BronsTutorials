To learn how to create a new Remote RADIUS Server Group, proceed with the following steps...
  • Begin first by right-clicking 'Remote RADIUS Server Groups'.
  • Then select 'New'. This will open the 'New Remote RADIUS Server Group' dialog box.
  • Once the 'New Remote RADIUS Server Group' dialog box is open, the first thing we need to do is give a name to our new Remote RADIUS Server Group.
  • In this case, we will name our new Remote RADIUS Server Group 'RADIUS Server Group'.
  • Now that we have our name defined, we next have to add a server to this group. To do this, press the 'Add' button.
  • Then add the name or the IP address of the server here.
  • In this case, we will add the name of our server which is 'nap'.
  • We can then verify if the name we entered can resolve to an IP address. To do this, press 'Verify'.
  • Then press 'Resolve'.
  • If the address is able to be resolved, then it will be displayed here.
  • Press 'OK'.
  • Next, select the 'Authentication/Accounting' tab.
  • On this tab there isn't much to configure, most of these settings can remain as is. The setting that does need to be configured is the 'Shared secret' setting.
  • The shared secret that is entered here must be the same shared secret that is entered on the client server.
  • Then enter the shared secret again to confirm.
  • Once the shared secret is entered, press 'OK'.
  • You can see here that our NAP server is now added to the remote RADIUS Server Group.
  • Press 'OK'.
