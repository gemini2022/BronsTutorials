To learn how to import NPS configuration, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Network Policy Server'...
  • Once in Network Policy Server, right-click 'NPS (Local)'.
  • Then select the option 'Import Configuration'.
  • Then select the file that contains the NPS configuration information, which in this case is 'NpsConfiguration'.
  • Once the file is selected, press 'Open'.
  • After pressing 'Open', a dialog box will appear informing you that the NPS server configuration was successfully imported.

  • Press 'OK'.
