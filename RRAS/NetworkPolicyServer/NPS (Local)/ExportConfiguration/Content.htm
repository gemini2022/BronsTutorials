To learn how to export NPS configuration, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Network Policy Server'...
  • Once in Network Policy Server, right-click 'NPS (Local)'.
  • Then select the option 'Export Configuration'.
  • Before the NPS configuration can be exported, you must confirm that you aware that all shared secrets will also be exported. To confirm this, select the option 'I am aware I am exporting all shared secrets'.
  • Then press 'OK'.
  • Next, we need to enter a name for the configuration file.
  • In this case, we will name the file 'NpsConfiguration'.
  • Once the name has been entered, press the 'Save' button to save the configuration.
