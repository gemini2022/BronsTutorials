To learn how to register the NPS server in Active Directory, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Network Policy Server'...
  • Once in Network Policy Server, right-click 'NPS (Local)'.
  • And select the option 'Register server in Active Directory'.
  • This will open a dialog box confirming if you want to proceed with authorizing this computer to read users dial-in properties.

  • Press 'OK'.
  • After pressing 'OK', you will receive another dialog box informing you that this computer is now authorized to read users dial-in properties.

  • Press 'OK' again.
  • Now that the NPS server is registered in Active Directory, there may be a situation were you need to unregister the NPS server from Active Directory. To do this, go back down to 'Start'.
  • Then go to 'Administrative Tools'.
  • And select 'Active Directory Users and Computers'.
  • Once in 'Active Directory Users and Computers', expand your domain, which in this case is 'bronstutorials.local'.
  • Then select the 'Users' OU.
  • Once the 'Users' OU is selected, in the window to the right, right-click the group 'RAS and IAS Servers'.
  • And select 'Properties'.
  • Once in the properties of the 'RAS and IAS Servers' group, select the 'Members' tab.
  • The member you are looking for is 'NPS'. When you registered NPS in Active Directory, NPS became a member of this group. To unregister NPS from Active Directory, make sure NPS is selected.
  • And press 'Remove'. Removing NPS as a member of this group will unregister NPS from Active Directory.
  • Once you press 'Remove', a dialog box will appear asking if you are sure you want to remove the selected member from this group.

  • Press 'Yes'.
  • After NPS has been removed from the 'RAS and IAS Servers' group, press 'OK'.
