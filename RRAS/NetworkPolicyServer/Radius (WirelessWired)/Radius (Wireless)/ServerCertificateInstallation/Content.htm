EAP-PEAPv0 (EAP-MSCHAPv2) requires a server certificate be installed on the RADIUS server in order to establish a secure TLS tunnel and to prove its identity to the client. To learn how to install a certificate on the RADIUS server, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then, in the search field, type 'mmc'.
  • After typing 'mmc' in the search field, select the 'mmc' icon located at the top of the menu. This will open up the MMC Console.
  • Once the MMC Console is open, select 'File'.
  • Then select 'Add/Remove Snap-in...'.
  • From here, we need to add the Certificates snap-in to the MMC Console. To do this, select 'Certificates'.
  • Then press the 'Add >' button.
  • Once we press the 'Add' button, we next need to choose what type of account we want to manage the certificates for. In this case, we want to choose the option 'Computer account'.
  • Press 'Next >'.
  • Then we need to specify which computer this snap-in will manage. Will this snap-in manage this computer, or will it manage another computer on the network? In this case, we want to manage this computer, so we will leave the default option selected 'Local computer'.

  • Press 'Finish'.
  • Now press 'OK' to create the snap-in.
  • Now that the Certificates snap-in is added to the MMC Console, we can now use the Certificates snap-in to request the Server Authentication Certificate we need and then install it. To do this, expand 'Certificates'.
  • Then right-click 'Personal'.
  • And go to 'All Tasks'.
  • Then 'Request New Certificate...', which will launch the 'Certificate Enrollment' wizard.
  • Once the 'Certificate Enrollment' wizard is open, press 'Next' to begin the certificate enrollment process.
  • Make sure 'Active Directory Enrollment Policy' is selected, and then press 'Next'.
  • On this next screen is a list of all the certificates that we can enroll for. In this case, the one that we want is 'Domain Controller'.
  • Once the 'Domain Controller' certificate is selected, press 'Enroll'.
  • Once the 'Enroll' button is pressed, the Certificate Enrollment wizard will begin the enrollment process.
  • Once the certificate has been successfully enrolled, press the 'Finish' button.
