To learn how to configure an access point for RADIUS authentication, proceed with the following steps...
  • Begin first by logging into your router or access point.
  • In most cases, before you can successfully log in, you'll need to provide a username and password.
  • In this case, for this particular router, the username is 'admin'.
  • And the password is 'admin'.
  • Once the username and password has been entered, press 'OK'.
  • The first thing we need to do once logged in, is go to the 'Wireless' tab.
  • Once in the 'Wireless' tab, make sure the wireless configuration option is set to 'Manual'.
  • Now that the wireless configuration is set to 'Manual', we are now given additional settings to configure. At this point, the first setting we want to take a look at is the SSID option. In this case, the default name for the SSID is set to 'linksys'. You can keep the default setting if you wish or you can change it to a more suitable name.
  • In this case, we will change the SSID to 'BronsTutorials'.
  • Next, make sure the the SSID is set to broadcast.
  • Then, once the appropriate settings have been configured, press the 'Save Settings' button.
  • After the settings have been saved, press the 'Continue' button.
  • Next, select the 'Wireless Security' tab.
  • Once in the 'Wireless Security' tab, select the dropdown.
  • Then select the option 'WPA2 Enterprise'.
  • Once the option 'WPA2 Enterprise' is selected, we next need to enter in the IP address of the RADIUS server that will be used for our RADIUS wireless authentication.
  • In this case, the address of the RADIUS server is '192.168.1.2'.
  • And then finally, enter in a shared key. In this case, the shared key we be '1234'.
  • Press the 'Save Settings' button to save the settings.
