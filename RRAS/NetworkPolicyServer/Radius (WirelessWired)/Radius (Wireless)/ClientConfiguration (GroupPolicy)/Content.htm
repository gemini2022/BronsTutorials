To learn how to configure wireless settings for a client using Group Policy, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Group Policy Management'...
  • Once in Group Policy Management, expand 'Forest'.
  • Then expand 'Domains'.
  • Then expand a domain, which in this case is 'bronstutorials.local'.
  • Once a domain has been expanded, right-click on a Group Policy, which in this case will be 'Default Domain Policy'.
  • Then select 'Edit...'. This will open the Group Policy Management Editor.
  • Once in the Group Policy Management Editor, expand 'Computer Configuration'.
  • Then expand 'Policies'.
  • And then expand 'Windows Settings'.
  • And then finally expand 'Security Settings'.
  • Under 'Security Settings', right-click on 'Wireless Network (IEEE 802.11) Policies'.
  • And then select the option 'Create A New Wireless Network Policy for Windows Vista and Later Releases'. This will open the properties of a new Wireless Network Policy.
  • The first setting is the policy name, which by defalut is 'New Wireless Network Policy'. You can leave this name as is, or change it to a more suitable name if you wish.
  • In this case, we will change the name to 'Wireless Policy'.
  • Next, we will change the description to 'Wireless policy for BronsTutorials network'.
  • Next, we need to create a network profile for this policy. To do this, press the 'Add...' button.
  • And select the option 'Infrastructure'. This will open the properties of a new network profile.
  • The first setting is the profile name, which by defalut is 'New Profile'. Again, you can leave this name as is, or change it to a more suitable name if you wish. However, it is recommended that this name be changed because this name is what will be broadcasted as the SSID. Even though there is an SSID setting that needs to be configured below, it is this setting oddly enough that will be broadcasted.
  • In this case, the name we will be 'BronsTutorials Network'.
  • Now the SSID we enter here, needs to be the same SSID setting used for the access point, which we configured earlier in a previous tutorial. So in this case, the SSID needs to be 'BronsTutorials'.
  • After the SSID has been entered, press the 'Add' button.
  • For this network profile, we will disable the option 'Connect automatically when this network is in range'.
  • Next, select the 'Security' tab.
  • Most of these settings on this tab can remain untouched. In this case we want to increase the the number of authentication failures. To do this, press here to increase the value.
  • In this case, we will increase the max authentication failures to '3'.
  • And next, we will disable the option 'Cache user information for subsequent connections to this network'. This will force the user the log in everytime a connection needs to be made.
  • To create the network profile, press the 'OK' button.
  • And to create the wireless policy, press 'OK'.
