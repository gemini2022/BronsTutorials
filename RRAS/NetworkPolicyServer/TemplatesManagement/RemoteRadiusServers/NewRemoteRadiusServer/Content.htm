To learn how to create a new Remote RADIUS Server template, proceed with the following steps...
  • Begin first by right-clicking 'Remote RADIUS Servers'.
  • And select 'New'.
  • The first thing we need to do is give a name to our Remote RADIUS Server template.
  • In this case, we will name our Remote RADIUS Server template 'Remote Radius Server'.
  • Next, enter the IP address of the RADIUS server.
  • Then press 'Next'.
  • On this screen there isn't much to configure, most of these settings can remain as is. The setting that does need to be configured is the 'Shared secret' setting.
  • The shared secret that is entered here must be the same shared secret that is entered on the client server.
  • Then enter the shared secret again to confirm.
  • Once the shared secret is entered, press 'Next'.
  • On this screen we can specify load balancing settings. In this case, we will leave the default settings and move on.

  • Press 'Next'.
  • And then finally, press 'Finish' to create the Remote RADIUS Server template.
