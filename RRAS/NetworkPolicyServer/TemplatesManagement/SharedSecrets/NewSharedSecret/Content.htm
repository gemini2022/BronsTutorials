To learn how to create a new Shared Secret template, proceed with the following steps...
  • Begin first by right-clicking 'Shared Secrets'.
  • And select 'New'.
  • The first thing we need to do is give a name to our Shared Secret template.
  • In this case, we will name our Shared Secret template 'Shared Secret'.
  • Then enter in a shared secret.
  • Then type the same shared secret again to confirm.
  • Once the shared secret is entered, press 'OK' to create the Shared Secret template.
