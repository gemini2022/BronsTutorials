To learn how to create a new RADIUS Client template, proceed with the following steps...
  • Begin first by right-clicking 'RADIUS Clients'.
  • And select 'New'.
  • The first thing we need to do is give a name to our RADIUS Client template.
  • In this case, we will name our Shared Secret template 'Radius Client'.
  • Then enter in the IP address of the RADIUS client.
  • Next, enter in a shared secret.
  • Then type the same shared secret again to confirm.
  • Once the shared secret is entered, press 'OK' to create the RADIUS Client template.
