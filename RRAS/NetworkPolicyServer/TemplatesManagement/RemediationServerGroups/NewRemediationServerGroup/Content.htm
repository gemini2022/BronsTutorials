To learn how to create a new Remediation Server Group template, proceed with the following steps...
  • Begin first by right-clicking 'Remediation Server Groups'.
  • And select 'New'.
  • The first thing we need to do is give our new Remediation Server Group template a name.
  • In this case, we will name our Remediation Server Group template 'Remediation Server Group'.
  • Next, we need to specify which servers on the network will be part of this remediation server group. To do this, press the 'Add' button.
  • The server that we will add will be our WSUS server. To begin, first we need to provide a friendly name for the server.
  • In this case, the name will be 'Remediation Server'.
  • Then enter in its IP address.
  • Once its IP address is entered, press the 'Resolve' button.
  • After the 'Resolve' button is pressed, its IP address will be displayed here.
  • Press 'OK' to add the server to the group.
  • You can see here that the remediation server has been added to the remediation server group.
  • Press 'OK' to create the Remediation Server Group template.
