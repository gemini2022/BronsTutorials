To learn how to create a new IP Filter template, proceed with the following steps...
  • Begin first by right-clicking 'IP Filters'.
  • And select 'New'.
  • The first thing we need to do is give our new IP Filter template a name.
  • In this case, we will name our IP Filter template 'IP Filter'.
  • Next, we can configure input and output filters for IPv4 here.
  • And we can configure input and output filters for IPv6 here.
  • Once the filters have been configured, press 'OK' to create the IP Filter template.
