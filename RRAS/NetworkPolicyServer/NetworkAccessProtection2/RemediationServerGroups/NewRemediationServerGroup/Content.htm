To learn how to create a new Remediation Server Group, proceed with the following steps...
  • Begin by right-clicking on 'Remediation Server Groups'.
  • Then select 'New'.
  • After we press 'New', the first thing we need to do is give a name to our new Remediation Server Group.
  • In this case, we will name our remediation group, 'Remediation Servers'.
  • Next, we need to specify which servers on the network will be part of this remediation server group. To do this, press the 'Add' button.
  • The first server that we will add will be our WSUS server. To begin, first we need to provide a friendly name for the server.
  • In this case, the name will be 'wsus'.
  • Then enter in the actual name of the server, which in this case is 'wsus'.
  • We can then verify the server by pressing the 'Resolve' button.
  • Once the 'Resolve' button is pressed, the IP address of the server will be placed here.
  • Press 'OK' to add the server to the group.
  • Another server that we want to add to the remediation server group is the Read Only Domain Controller server. To do this, press the 'Add' button again.
  • And then type in the friendly name for this server, which in this case will be 'rodc'.
  • Then type the actual name of the server, which is 'rodc'.
  • verify the name.
  • And then press 'OK'.
  • Once you have the group defined, press 'OK' to create the group.
