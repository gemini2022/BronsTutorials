To learn how to configure accounting, proceed with the following steps...
  • Begin first by selecting the 'Configure Accounting' link located in the window to the right. This will open the 'Accounting Configuration Wizard'.
  • Once the 'Accounting Configuration Wizard' is open, press 'Next' to begin the accounting configuration process.
  • On this screen, you are presented with four options on how you would like to store you accounting data. The first option is logging to a SQL Server database.
  • The next option is to just store the data to a text file on the local computer.
  • Or, you could choose the option to log the data on both a SQL Server database and to a local text file.
  • And then finally, you have the option to log the data to a SQL Server database, but also the option to log to a text file if the SQL logging fails.
  • In this case, we don't have a SQL Server database, therefore we will choose the option 'Log to a text file on the local computer'.
  • Press 'Next'.
  • On this next screen of the wizard, we need to configure what type of information we will be logging. First is the option to log 'Accounting Requests'.
  • We can also choose to log 'Authentication requests'.
  • Next, we have the option to log 'Periodic accounting status'.
  • And then finally, we can log 'Periodic authentication status'.
  • This section displays the directory where the log file will be located. You can choose a different location if you wish or keep the default.
  • And then finally, you have the option to discard connection requests if for any reason logging fails due to network or other problems.

  • Press 'Next'.
  • And then finally, review the settings you have configured and then press 'Next'.
  • Accounting is now configured! Press 'Close'.
