To learn how to configure a client computer to connect to a workplace using a PPTP VPN connection, proceed with the following steps...
  • Begin first by selecting the network icon at the bottom right corner of your screen.
  • Then select 'Open Network and Sharing Center'. This will open the 'Network and Sharing Center' screen.
  • Once in 'Network and Sharing Center', select the option 'Set up a new connection or network'. This will bring us to the 'Set Up a Connection or Network' screen.
  • Once at the 'Set Up a Connection or Network' screen, we have to decide what kind of connection we want to make. In this case, we want to make a remote connection to the network of our workplace. To do this, select the option, 'Connect to a workplace'.
  • Then click 'Next'.
  • Next, we have to decide how the connection to the workplace will be made. In this case, we will be connecting via a VPN connection. To do this, select the option 'Use my Internet connection (VPN)'.
  • Once the option 'Use my Internet connection (VPN)' is selected, you will then need to specify the internet address of the workplace you are connecting to. In this first field, you can either enter in the IP address...
  • Or you can do what we have demonstrated here which is, enter in the name of the VPN server. However, if you do decide to take this approach, you need to make sure that your corporate VPN server has a publicly resolvable host name.
  • This next field specifies the name of the connection. Windows provides a name for you, but you have the option to change it if you wish. In this case, we will leave the default name and move on.
  • Select this option if you are using a smart card for authentication. In this case, we will not be using a smart card for this example.
  • If you want to allow anyone who is using the computer to use this connection, then select this option.
  • And then finally, select the option 'Don't connect now; just set it up so I can connect later'. Selecting this option will prevent the connection from automatically being made when the configuration of the VPN connection is complete. We want this option selected because there are additional settings that will need to be configured before we can make a connection.
  • Press 'Next'.
  • On this screen we need to provide some login credentials for the VPN connection. In this first field, we need to enter in the name of a user that has been granted remote access permissions.
  • In this case, the user that we will use for this VPN connection will be the administrator.
  • Once the name of the user has been entered, type in the password.

  • If the option 'Show characters' is selected.
  • Then the password you enter will be revealed.
  • This next option, 'Remember this password', allows the password that you entered to be stored. This prevents the need to enter in a password the next time a VPN connection is made.
  • And then finally, you have the option to enter in the name of the domain you are connecting to. This setting is optional, so we will leave this field left blank and move on.
  • Press the 'Create' button.
  • After the 'Create' button is pressed, a VPN connection to your workplace can be made. However, there are still a few more settings that need to be configured. To configure these settings, first press the 'Close' button.
  • Then select the option 'Change adapter settings'. This will bring you to the 'Network Connections' screen.
  • Once in 'Network Connections', you will notice that there are two network connections present.
  • The first connection labeled 'Local Area Connection', is the connections that is connected to the network adapter installed on this machine.
  • This next connection is the VPN connection that we just created.
  • With this connection, take notice of the type of VPN protocol that this connection is configured for. By default, when a VPN connection is first created, it randomly chooses the type of VPN protocol that will be used.
  • In this case, the type of VPN that we want to demonstrate for this example is PPTP. To change this, right-click on the VPN Connection.
  • And select properties.
  • Once in the properties of the VPN connection, select the 'Security' tab.
  • Here on the Security tab with the first setting, is where we change the type of VPN connection from a random automatic setting to the type of VPN connection we want.
  • To do this, select the dropdown.
  • And choose the desired setting, which in this case is 'Point to Point Tunneling Protocol (PPTP)'.
  • Once the desired type of VPN option is selected, press 'OK'.
  • You can see that the protocol we want to use for our VPN connection is now configured to our desired protocol.
  • And that now we can connect to the workplace. To do this, right-click on the VPN connection.
  • And select 'Connect'.
  • Before the connection is made, the proper credentials have to be provided, but since we configured this VPN connection to remember our user name and password, this section can be left untouched. However, these settings could be updated if needed.
  • And then finally, if we had decided earlier not to have our user name and password saved, we now have the option in this section to save it here.
  • Press the 'Connect' button to connect.
