To learn how to configure the Routing and Remote Access server for VPN connections using the SSTP protocol, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, right-click the server node, which in this case is 'RRASSERVER (local)'.
  • Then select 'Properties'.
  • Once in the RRASSERVER properties, select the 'Security' tab.
  • At the bottom of the Security tab there will be a section that pertains to the selecting of the certificate that will be used for SSTP server authentication.
  • To select this certificate, select the dropdown.
  • And choose a certificate in the list. In this case, we will select the certificate named 'rrasserver.bronstutorials.com'.
  • Once the certificate is selected, press 'OK'.
  • After you press 'OK', you will be informed that in order for the changes that you just made to take effect, the Routing and Remote Access service must be restarted.

  • Press 'Yes' to restart the Routing and Remote Access service.
  • Once you press 'Yes', the Routing and Remote Access service will stop.
  • Then the Routing and Remote Access service will start back up.
  • And then finally, the Routing and Remote Access service will finish initialization.
