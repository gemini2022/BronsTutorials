To learn how to get to the RRAS Server properties, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, right-click the server node, which in this case is 'RRASSERVER (local)'.
  • Then select 'Properties'.
