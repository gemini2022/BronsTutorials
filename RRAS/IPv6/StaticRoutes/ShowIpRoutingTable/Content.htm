To learn how to view the IP Routing Table, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'RRASSERVER (local)'.
  • Then expand 'IPv6'.
  • Once you expand 'IPv6', right-click 'Static Routes'.
  • Then select the option 'Show IP Routing Table...'.
  • Once you select 'Show IP Routing Table...', a window will appear displaying information about all the IP routing.
