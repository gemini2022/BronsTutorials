Static routes informs the router of a specified network, the address of the gateway that serves as the next hop, and the interface to use to forward packets sent to that network. To learn about static routes, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'RRASSERVER (local)'.
  • Then expand 'IPv6'.
  • Once you expand 'IPv6', select 'Static Routes'.
  • Once you select 'Static Routes', in the window to the right side of the Routing and Remote Access screen, you will see a list of details that pertain to all the static routes that have been configured. In this case, there are no static routes configured, but if there were, they would be listed here.
