To learn how to create a new routing protocol, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'RRASSERVER (local)'.
  • Then expand 'IPv6'.
  • Once you expand 'IPv6', right-click 'General'..
  • Then select 'New Routing Protocol'.
  • Once you select 'New Routing Protocol', you then need to choose the routing protocol you want to add. In this case there is only one option. Once the routing protocol is selected, press 'OK'.
  • Pressing 'OK' will create the 'DHCPv6 Relay Agent' node found under the 'IPv6' node.
