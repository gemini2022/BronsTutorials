To learn about the Servers tab, proceed with the following steps...
  • First on this tab, we have to configure the address of the DHCP server that this DHCP relay agent will send DHCP requests to. In this field labeled 'Server address', is where the address of the DHCP server is entered.
  • In this case, the address of the DHCP server is '2002:c0a8:c832:0:0:0:0:0'.
  • Once the address of the DHCP server has been entered, press the 'Add' button.
  • You may add more DHCP server addresses if needed, otherwise press 'OK'.
