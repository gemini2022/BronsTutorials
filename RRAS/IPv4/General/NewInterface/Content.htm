To learn how to create a new interface, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'RRASSERVER (local)'.
  • Then expand 'IPv4'.
  • Once you expand 'IPv4', right-click 'General'..
  • Then select 'New Interface'.
  • Once you select 'New Interface', you then need to choose the interface you want to add. In this case there is only one option. Once the interface is selected, press 'OK'.

  • Pressing 'OK' will open up the properties of the new interface.
