To learn about the Preference Levels tab, proceed with the following steps...
  • This tab gives you a way to change the router's behavior by telling it what class of routes to prefer. Routers select routes based on cost information, but another factor that comes into play is the preference level of the routing source. The default configuration for RRAS causes it to prefer local and static routes over dynamically discovered routes. If it can't use a local or static route, it will try to use a RIP generated route instead.

  • You can change the router's class preference by selecting the class you want to change and using the 'Move Up' and 'Move Down' buttons.
