To learn about the NAT tab, proceed with the following steps...
  • This first option specifies that this interface is a private interface that is connected to the private network.
  • This next option specifies that this interface is the public interface that is connected to the Internet.
  • And then finally, this last option specifies whether the router performs TCP port and UDP port translation in addition to IP address translation. If you only have a single public IP address, you must select this check box. Otherwise, network traffic from hosts on the private network cannot be properly translated to Internet traffic.
