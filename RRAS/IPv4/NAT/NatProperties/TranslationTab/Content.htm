To learn about the Translation tab, proceed with the following steps...
  • This first setting, 'Remove TCP mapping after (minutes)', represents the number of minutes that a dynamic mapping for a TCP session remains in the translation table for the router. You can also click the arrows to select a new setting. The default value for TCP sessions is 1,440 minutes (24 hours).
  • This next setting, 'Remove UDP mapping after (minutes)', represents the number of minutes that a dynamic mapping for a UDP message remains in the translation table for the router. You can also click the arrows to select a new setting. The default value for UDP messages is 1 minute.
  • And then finally, pressing the button 'Reset Defaults', will reset the time-out values for dynamic mappings for TCP traffic to 1,440 minutes (24 hours) and UDP traffic to 1 minute.
