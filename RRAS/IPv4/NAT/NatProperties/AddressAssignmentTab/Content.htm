To learn about the Address Assignment tab, proceed with the following steps...
  • This first option, 'Automatically assign IP addresses by using the DHCP allocator', specifies whether the router provides DHCP-based address assignment to DHCP clients on the private network.
  • This next setting, 'IP address', provides a space for you to type an Internet Protocol (IP) address that corresponds to the range of addresses to be assigned to DHCP clients on the private network. The network you configure here should be a private network or a subnet of a private network. The private networks are 10.0.0.0 with a subnet mask of 255.0.0.0, 172.16.0.0 with a subnet mask of 255.240.0.0, and 192.168.0.0 with a subnet mask of 255.255.0.0. The default IP network is 192.168.0.0.
  • This setting, 'Mask', provides a space for you to type the subnet mask that corresponds to the range of addresses to be assigned to hosts on the private network. The default subnet mask is 255.255.255.0.
  • You have the option to specify any addresses within the configured range of addresses that should not be assigned to DHCP clients on the private network because they are already in use. To do this, begin by first pressing the 'Exclude...' button.
  • Once the 'Exclude' button is pressed, a dialog box will appear that lists all the excluded addresses.
  • To add an address to this list, begin by first pressing the 'Add..' button.
  • Once the 'Add' button has been pressed, a dialog box will appear.
  • Enter the address that you want to add to the exclude list. In this case, the address we will enter is '192.168.0.10'.

  • Press 'OK'.
