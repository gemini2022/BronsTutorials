To learn how to view RIP neighbors, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'RRASSERVER (local)'.
  • Then expand 'IPv4'.
  • Once you expand 'IPv4', right-click 'RIP'.
  • Then select the option 'Show Neighbors...'.
  • From here you will be able to see all the RIP neighbors that have been detected.
