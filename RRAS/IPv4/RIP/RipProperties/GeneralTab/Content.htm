To learn about the General tab, proceed with the following steps...
  • This first setting, 'Maximum delay', specifies the interval between triggered updates. Triggered updates are sent when the entries in a routing table change. The default is 5 seconds.
  • This option, 'Log errors only', specifies that only errors are logged in the System Log in Event Viewer.
  • This next option, 'Log errors and warnings', specifies that errors and warnings are logged in the System Log in Event Viewer.
  • Next, this option 'Log the maximum amount of information', specifies that the maximum amount of information is logged in the System Log in Event Viewer.
  • And then finally, this last option, 'Disable event logging', specifies that no events are logged in the System Log in Event Viewer.
