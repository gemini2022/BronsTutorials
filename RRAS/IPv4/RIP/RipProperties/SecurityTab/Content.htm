To learn about the Security tab, proceed with the following steps...
  • This first option, 'Accept announcements from all routers', specifies that announcements from all routers are accepted. This is the default.
  • This next option, 'Accept announcements from listed routers only', specifies that only announcements from the routers listed are accepted. Any announcements from routers that are not on the list are ignored. This option is useful if you want to accept RIP announcements only from specific RIP routers.
  • Once the option 'Accept announcements from listed routers only' is selected, you will then need to provide the IP address of the router you want to accept announcements from in this section here.
  • In this case, the IP address of the router we want to accept announcements from is '192.168.100.1'.
  • Once the IP address is entered, press the 'Add...' button.
  • Pressing the 'Add' button will add the IP address of the router to this list.
  • This option, 'Ignore announcements from all listed routers', specifies that announcements from the routers listed are ignored. Announcements from all other routers are accepted. This option is useful if you want to exclude the announcements of specific RIP routers.
  • Once the option 'Ignore announcements from all listed routers' is selected, you will then need to provide the IP address of the router you want to ignore announcements from. Any routers added to this list will now be ignored.
