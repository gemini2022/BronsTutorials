To learn how to get to the properteis of an organizational unit, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then select 'Active Directory Users and Computers'...
  • Once in 'Active Directory Users and Computers', select 'View' on the top menu of the 'Active Directory Users and Computers' window.
  • Then select 'Advanced Features'. This will unhide any additional OU properties that are normally hidden.
  • Then, expand your domain, in this case it's 'bronstutorials.local'.
  • Now, right-click on an OU that Group Policy can be appied to. This would be an OU that you created or the 'Domain Controllers' OU. In this case, we're going to right-click on the 'New York' OU that we created earlier.
  • Then select 'Properties'.
