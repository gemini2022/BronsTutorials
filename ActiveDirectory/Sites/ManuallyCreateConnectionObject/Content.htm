The KCC Automatically creates connection objects in Active Directory. However, users can manually create connection objects to customize the network’s topology or decrease the number of hops from one domain controller to another particular domain controller. When the KCC creates connection objects, the KCC automatically removes them when the replication topology changes. Connection objects that are manually created are not removed when the replication topology changes. Users have to manually remove these connection objects. To learn how to manually create a connection object, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then select 'Active Directory Sites and Services'...
  • Once in 'Active Directory Sites and Services', expand 'Sites'.
  • Then expand the site you want to make the connection object from. In this case we're going to expand the 'NewYork' site.
  • Then expand the 'Servers' container.
  • Once in the 'Servers' container, expand the domain controller you want to make the connection object from. In this case, we're going to expand the 'NYDC1' domain controller.
  • Then right-click on 'NTDS Settings'.
  • And select 'New Active Directory Domain Services Connection...'.
  • At the bottom of the form, select the domain controller you want to make a connection object to. In this case, we're going to select the 'WASHDC1' domain controller.

  • Then press 'OK'.

 
  • Once you press 'OK' on the previous form, a message will appear if you already have a connection made from the source server to the destination server.

  • Press the 'Yes' button to continue.
  • At this point, you are to choose a name for your connection object. Windows will generate a name for you. You may use the generated name they come up with or simply make your own. We're going to go with the generated name and move on.

  • Press 'OK' to create the connection object.
  • Once the connection object is made, if you look to the far right side of your screen, you will see the connection object that you have just made within the list of the other <automatically generated> connections.
