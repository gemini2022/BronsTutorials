To learn how to protect and object in 'Active Directory Sites and Services' from getting moved or accidently deleted, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then select 'Active Directory Sites and Services'...
  • Once in 'Active Directory Sites and Services', expand 'Sites'.
  • Then go to the 'NewYork' site and expand that.
  • Then expand 'Servers'.
  • In this case, we want to make sure that 'NYDC1' does not get deleted or moved. To do this, right click on the object you want to prevent accidental deletion, in this case, 'NYDC1'.
  • Then select 'Properties'.
  • Once in the properties of 'NYDC1', go to the 'Object' tab.
  • It is here where you will select the option 'Protect object from acciental deletion'.

  • Then press 'OK'.
