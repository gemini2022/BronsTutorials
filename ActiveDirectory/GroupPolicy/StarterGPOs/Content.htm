To learn how to create a Starter GPO, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then select 'Group Policy Management'...
  • Once in 'Group Policy Management', Select 'Starter GPOs'.
  • Starter GPOs are not configured by default. To start using Starter GPOs, select the button at the far right side of the window, 'Create Starter GPOs Folder'.
  • You will notice that once this option is selected, a number of Starter GPOs are created.
  • If you wanted to create a new Starter GPO, you can right-click on 'Starter GPOs', back at the left side of the window...
  • And select 'New'.
  • Once you select 'New', a 'New Starter GPO' form will appear.
  • To create the Starter GPO, all you need to do is first enter in a name. In this case we're going to call this GPO, 'Default Branch Office'.

 
  • You can then also enter in a comment if you wish.

  • Once complete, click 'OK'.
  • Once the Starter GPO is created, you can then right-click on it...
  • And select 'Edit...'.
  • After selecting 'Edit...', a 'Group Policy Starter GPO Editor' will appear. You will notice that Starter GPOs only have 'Administrative Templates'. You can not configure any other setting in a Starter GPO.
  • For this demonstration, we will expand 'Administrative Templates' under the 'User Configuration' section.
  • Then open 'Desktop'.
  • Then select the 'Desktop' settings.
  • Once in the 'Desktop' settings, open the setting, 'Desktop Wallpaper'.
  • We'll configure this setting by first selecting the 'Enabled' option.
  • Then entering in the path to the wallpaper image we want to use for this setting.
  • And then finally, selecting the dropdown.
  • And choosing the 'Stretch' option.
  • When finished, press 'OK'.
  • Once complete, the next step is to go back into 'Group Policy Management' and create a new Group Policy object.

  • Close this 'Group Policy Starter GPO Editor' form.
  • Now we will create a new Group Policy under 'New York'. To do this, we'll right-click on 'New York'.
  • And select, 'Create a GPO in this domain, and link it here...'.
  • After selecting the, 'Create a GPO in this domain, and link it here...' option, a 'New GPO' form will appear.
  • For this new GPO, we will call it 'New York Office'.
  • Next, if we select the dropdown option...
  • You'll notice all the Starter GPOs, plus the new Starter GPO we just created. For this example, we will select our new Starter GPO, 'Default Branch Office.
  • Then click 'OK'.
  • If we now open the 'New York' OU...
  • You will see that the Group Policy has been created and linked here under the 'New York' OU using the Starter GPO as the template.
