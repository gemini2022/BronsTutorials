To learn about the Group Policy Details tab, proceed with the following steps...
  • The first detail displays the name of the domain. In this case, it displays 'bronstutorials.local'.
  • The owner of this Group Policy is displayed here.
  • Next, is a time stamp of when the Group Policy was created.
  • And here is a time stamp of when this Group Policy was last modified.
  • This displays the versions between both the Active Directory replication version and the SysVol replication version for the user side of Group Policy. As long as both version numbers are the same, Group Policy is in sync.
  • And this displays the versions between both the Active Directory replication version and the SysVol replication version for the computer side of Group Policy. As long as both version numbers are the same, Group Policy is in sync.
  • Here you can see the unique id for this group policy. This is used to reference the Group Policy Object in Active Directory, in the SysVol folder, and the Active Directory Database.
  • Here we have the ability to set the status of this GPO. If we select the drop down...
  • The first option allows us to disable this GPO.
  • The next option allows us to disable only the computer side of this Group Policy.

 
  • Here is where we can enable this Group Policy. Both the user and computer side of this group policy will be enabled.
  • And finally, this last option allows us to disable only the user side of this Group Policy.
  • The last detail allows us to see any comments that have been written for this Group Policy. To leave a comment for a Group Policy, simply edit the policy and then right click on the policy name in the Group Policy Management Editor and select Properties. The last tab is where you can leave a comment.
