To learn how to copy GPO's to another domain, proceed with the following steps...
  • If we begin by selecting the 'Advanced...' button...
  • We can see what type of permissions these groups and users have for this Group Policy.
  • If we select a group or user, in this case well select the group 'GSales_Staff', and scroll down...
  • We can see that the permissions that this group has been delegated for this Group Policy is 'Read' and 'Apply group policy'. In oder for a Group Policy to be applied, both these settings must be applied.
