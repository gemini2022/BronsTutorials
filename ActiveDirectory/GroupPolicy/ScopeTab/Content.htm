To learn about the Group Policy Scope tab, proceed with the following steps...
  • The first section allows for setting the location of where the group policy will be displayed. For example, if we select the dropdown...
  • We can see that we have the option of displaying the Group Policy in the 'Entire Forest'...
  • 'All Sites'...
  • Or just the domain, which in this case is 'bronstutorials.local'.
  • Here in this section, you will see all the OUs this Group Policy is applied to. In this case, only one OU is applied, which hapeens to be entiere domain 'bronstutorials.local'. If this Group Policy was applied to other OUs, those OUs would be displayed in this list.
  • Here, this section determines what users and computers this Group Policy will be applied to. At present, the group that this Group Policy is being applied to is 'Authenticated Users'. Authenticated users means any users and computers that where authenticated by the domain controller succesfully.
  • But for this example, we want to be more specific as to what group this Group Policy is being applied to, so we will remove this group. To do this, we will press the 'Remove' button.
  • Then press 'OK' to confirm the removal of the group.
  • Now we want to add a group. To do this, press the 'Add...' button.
  • Here we will enter in the group that we want. In this case we will enter the group 'gsales_staff'.

 
  • This Group Policy will now apply to only users and computers that are members of this group. This is a good way of narrowing down which users and computers this Group Policy will have an effect on.
  • This section is for the case where you may not be able to use groups to narrow down or target particular users or computers for a Group Policy. For example, perhaps you want to target particular computers that are running a particular operating system. To do this, you can link a WMI filter to the Group Policy. If we select this dropdown...
  • You will see that there are no WMI filters to select from, so we will need to create one first.
  • To do this, right-click 'WMI Filters' back at the left side of the window.
  • Then select the option 'New'.
  • Once you select 'New', a 'New WMI Filter' form will appear.
  • For this WMI Filter, we will create a filter that will check for computers that are running Windows 7. Therefore, we will call this WMI Filter 'Windows 7'.
  • And then give it a brief description.
  • The next step is to then click on the 'Add' button.
  • Once you select 'Add', a 'WMI Query' form will appear.
  • The information we will be adding is actually standard SQL. In short, the query is selecting any computer that is running Windows 7.

  • When finished, press 'OK'.
  • Press the 'Save' button to create the WMI Filter.
  • Now if we select the dropdown...
  • We now have a WMI Filter to choose from, which is the WMI Filter we just created. Select the WMI Filter to continue.
  • And then finally, press the 'Yes' button to confirm the changes and commit to the selected WMI Filter.
