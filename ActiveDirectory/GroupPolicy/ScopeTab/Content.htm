To learn how to copy GPO's to another domain, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then select 'Group Policy Management'...
  • Once in 'Group Policy Management', Expand 'Group Policy Objects'.
  • And right-click the Group Policy Object you want to copy. In this case, we're going to right-click the 'New York Office' Group Policy Object.
  • And select the option 'Back Up...'.
  • Once you select the 'Back Up...' option, you'll need to provide a location as to where the backup copy of the GPO will be placed. In this case, we will backup the GPO to the desktop. To do this, we will first click on 'Browse...'.
  • Then select 'Desktop'.
  • And press 'OK'.
  • Once the location of the backup is defined, you have the option to add a brief description.

 
  • Once complete, press the 'Back Up' button.
  • The backup process does not take long.
  • Once complete, press 'OK'.
  • The next step is to import the settings. For this example, we will be copying the GPO to the same domain, but in a real situation, you would more than likly be copying the GPO to another domain. Since the settings will be imported, it is a good idea to create a new Group Policy. If you use an existing Group Policy Object, you will merge the backup settings with the settings that exist in the Group Policy. In this case we will create a new Group Policy Object by right-clicking 'Group Policy Objects'...
  • And selecting 'New'.
  • Once you select 'New', a 'New Starter GPO' form will appear.
  • In this case, we will name the Group Policy Object, 'LA Office'.

  • Then press 'OK'.
  • Now that the Group Policy Object is created....
  • We can then right-click it...
  • And select 'Import Settings...'.
  • Once in the 'Import Settings Winzard', you can skip pass the welcome screen by pressing 'Next'.
  • The next screen gives you the option to backup the GPO before starting the import. This is a good idea if you are merging the settings with an existing Group Policy. In this case, we are importing the settings into a brand new empty Group Policy, so there is really no need to backup the settings.

  • Click 'Next' to continue.
  • On the next screen, we need to give the path to where we backed up the previous Group Policy. To do this, we will click 'Browse'.
  • Select 'Desktop'.
  • And press 'OK'.
  • Once the path to the backup is defined, press 'Next'.
