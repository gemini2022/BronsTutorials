DCdiag is another command-line tool that can be used to find out which domain controller is holding which operation master role. To learn how to use Dcdiag, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then open the 'Command Prompt'.
  • Once the command prompt is open, type in the command 'dcdiag /test:knowsofroleholders /v'. The '/v' switch, which stands for verbose, is added at the end of the command to give additional information.
  • After you run the command, you'll have to scroll up to find the information you are looking for.
  • Once you scroll up, you'll find listed are all five of the operation master roles and which domain controllers they are currently located on.
