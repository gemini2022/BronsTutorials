To remove a child domain, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then type, 'dcpromo' in the search field.
  • After, run the 'dcpromo' command by clicking on 'dcpromo' at the top of the menu.
  • Windows will then check if 'Active Directory Services' binaries are installed.
  • The first screen of the wizard is the 'Welcome Screen'. This screen explains that this wizard will uninstall 'Active Directory Domain Services'

  • Click on the 'Next' button to continue.
  • The next screen explains that uninstalling 'Active Directory Domain Services' will remove this server as a global catalog server and that you should insure that there is at least one other global catalog server in the domain.

  • Click on the 'OK' button to continue.
  • In this next screen, you are asked to indicate whether this is the last domain controller in the domain. In this case, this will be the last domain controller in the domain, so we will check this checkbox.

  • Click on the 'Next' button to continue.
  • In this screen, you are asked to specify an account that is a member of the Enterprise Admins group in the forest. Since the administrator that is currently logged in is not a member of the Enterprise Admins group, we will first select the 'Alternate credentials' option.
  • Then click on the 'Set...' button.
  • Once the 'Set..' button has been clicked, a little dialog box will appear. It is here where you will provide a user name and password of an account that is a member of the 'Enterprise Admins' group. In this case, it is the administrator of the root doamin.

  • Once the account credentials has been specified, click 'OK'.
  • Click on the 'Next' button to continue.
  • Next, provide a password for the new administrator account on this server. This essentially means that the original local administrator account will be replaced with a new administrator account and that the password that you provide on this screen will be the password for the new account.

  • Click on the 'Next' button to continue.
  • This screen of the wizard expalins that this domain will no longer exist once the uninstall process is complete.

  • Click on the 'Next' button to continue.
  • Once you press 'Next' on the previous screen, Active Directory Domain Services will begin removing the domain.
  • The last screen of the wizard explains that 'Active Directory Domain Services' was removed from this computer but that the 'Active Directory Domain Services' binaries are still installed. Removing the 'Active Directory Domain Services' binaries will be covered in a later tutorial, for now click 'Finish' to complete the wizard.
  • Once the wizard is complete, click on the 'Restart Now' button to restart the server so that the changes made by the Active Directory Domain Services wizard can take effect.
