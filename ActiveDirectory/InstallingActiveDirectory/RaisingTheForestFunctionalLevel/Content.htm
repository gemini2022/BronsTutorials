In Active Directory, domain controllers can run different versions of Windows Server operating systems. The functional level of a forest depends on which versions of Windows Server operating systems are running on the domain controllers in the forest. The functional level of a forest controls which advanced features are available in the forest. To raise the forest functional level, proceed with the following steps:
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then select 'Active Directory Domains and Trusts'...
  • Right click on 'Active Directory Domains and Trusts' found at the top left side of your screen.
  • Then select the option 'Raise Forest Functional Level...'.
  • In this case, the current forest functional level is set to Windows Server 2003.
  • To raise the forest functional level, simply click on the dropdown menu and select the forest functional level you want to raise it to.
  • Then click on the 'Raise' button.
  • Windows will then give you a warning reminding you that the change can NOT be reversed and will affect all the domains in the forest.

  • Click on the 'OK' button to commit to the changes.
  • Once the 'OK' button in the previous prompt has been clicked, another prompt will appear telling you that the raise was successful and that replication to each Active Directory Domain Controller may vary depending on your replication topology.

  • Click on the 'OK' button.
