In Active Directory, domain controllers can run different versions of Windows Server operating systems. The functional level of a domain depends on which versions of Windows Server operating systems are running on the domain controllers in the domain. The functional level of a domain controls which advanced features are available in the domain. To raise the domain functional level, proceed with the following steps:
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then select 'Active Directory Users and Computers'...
  • Right click on your domain found at the top left side of your screen. In this case, it's 'bronstutorials.local'.
  • Then select the option 'Raise Domain Functional Level...'.
  • Here, the current domain functional level is set to Windows Server 2003.
  • To raise the domain functional level, simply click on the dropdown menu and select the domain functional level you want to raise it to.
  • Then click on the 'Raise' button.
  • Windows will then give you a warning reminding you that the change can NOT be reversed and will affect the entire domain.

  • Click on the 'OK' button to commit to the changes.
  • Once the 'OK' button in the previous prompt has been clicked, another prompt will appear telling you that the raise was successful and that replication to each Active Directory Domain Controller may vary depending on your replication topology.

  • Click on the 'OK' button.
