The 'Personal Virtual Desktop' tab lets you assign a virtual Hyper-V machine to the user when using 'Remote Desktop'
  • To begin, first select the 'Assign a personal virtual desktop to this user' option.
  • Then click on the 'Browse...' button.
  • At this point, enter in the name of the server that is hosting 'Hyper-V'.

  • Then press 'OK'.
  • Once finished, press 'OK'.
