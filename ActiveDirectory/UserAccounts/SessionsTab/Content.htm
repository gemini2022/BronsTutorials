The 'Sessions' tab allows you to configure some options for the session created by 'Remote Desktop Services'.
  • The time taken for a disconnected session to be ended can be configured here. To do this, first select the dropdown button...
  • And select the time it will take before a disconnected session will end.
  • Here you have the option to configure how log the session can be running. This stops a user from staying connected indefinitely. To configure this, first select the dropdown button...
  • Then select how long this session will run for.
  • This option allows an idle limit to be configured. This defines how long a user can be connected to the server without doing anything. To configure this, first select the dropdown button...
  • Then select how long the idle limit will be.
  • Here, you have a setting that determines what will happen when one of the two session limits above are reached. By default the user will be disconnected, and they can then reconnect and carry on from where they left off. The other option ends the session.
  • This setting allows the reconnection options to be configured. When a session is reconnected, you can configure it so the user has to reconnect to the same computer they originally connected with or they can reconnect from any computer that they want.
