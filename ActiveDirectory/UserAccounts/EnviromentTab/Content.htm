On the 'Enviroment' tab, these settings are used when connecting to 'Remote Desktop Services'.
  • The first setting allows a program to run automatically when the user first logs in. To configure this, first select the 'Start the following program at logon' option.
  • Then type in the program you want to run. In this example, we're going to run Microsoft's paint program, 'mspaint.exe'.
  • Then provide the start in location. The purpose of this field is for when the program gets saved, it will be saved in this location.
  • At the bottom, the devices that are available to the client can be configured. The option allows drivers and printers to be connected when the user connects and changes the default printer used in the session to the client local printer.
