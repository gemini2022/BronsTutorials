On the 'Telephones' tab, you can configure additional contact information about the user.
  • On the first field, you can enter in the home phone number of the user.
  • If the user has more than one home phone number, you can add more by first selecting the 'Other...' button...
  • Then enter in another home phone number and press 'Add'.
  • If the user has more, you can continue to add more if you wish, otherwise, press 'Ok'.
  • Here you can enter in the user's pager number.
  • If the user has more than one pager number, you can add more by first selecting the 'Other...' button...
  • Then enter in another pager number and press 'Add'.
  • If the user has more, you can continue to add more if you wish, otherwise, press 'Ok'.
  • Now you can enter in the user's mobile number.
  • If the user has more than one mobile number, you can add more by first selecting the 'Other...' button...
  • Then enter in another mobile number and press 'Add'.
  • If the user has more, you can continue to add more if you wish, otherwise, press 'Ok'.
  • Then you can enter in a fax number.
  • If the user has more than one fax number, you can add more by first selecting the 'Other...' button...
  • Then enter in another fax number and press 'Add'.
  • If the user has more, you can continue to add more if you wish, otherwise, press 'Ok'.
  • Finally, you can enter in an IP phone.
  • If the user has more than one IP phone, you can add more by first selecting the 'Other...' button...
  • Then enter in another IP phone number and press 'Add'.
  • If the user has more, you can continue to add more if you wish, otherwise, press 'Ok'.
  • In the last field, you can enter in notes concerning contact information.

  • Press 'OK' when finished.
