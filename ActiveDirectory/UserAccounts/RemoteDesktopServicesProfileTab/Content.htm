The 'Remote Desktop Services Profile' tab is similar to the 'Profile' tab, but refers to storing a user profile for a 'Remote Desktop' session.
  • Like the 'Profile' tab, this field asks for the location of where to store the user's profile.
  • Also like the 'Profile' tab, you can define a location as to where the user will store their documents. Here you can either enter in a location...
  • Or select the connect option...
  • Then select the drive dropdown button.
  • Scroll to your desired drive letter.
  • Then select a drive of the location.
  • And then the path.
  • Finally, you can specify if you want this user to be denied permissions to log on to 'Remote Desktop'.

  • Press 'OK' when finished.
