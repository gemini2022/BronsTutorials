On the 'Organization' tab, you can once again configure additional information about the user.
  • Here you can enter in the user's job title.
  • And then, enter in the department where the user works in.
  • Then the company where the user works at.
  • You can provide the user's manager. This is useful if you want to know who this user reports to. To configure this, first select the 'Change...' button.
  • Then type in the name of the user's manager.

  • When finished, press 'OK'.
  • The last field represents the people that report to this user. In other words, the people listed in the 'Direct Reports' field have defined this user as their manager. Therefore, this field is not editable.

  • Press 'OK' when finished.
