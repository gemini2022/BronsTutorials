On the 'Address' tab, you can configure additional information about the user.
  • Start by adding the street of the user.
  • If the user has a P.O Box, then you can enter it here.
  • Enter the city of the user.
  • And the state where the user lives.
  • Then enter in the zip code, or postal code.
  • To enter in the country or region, select the dropdown button.
  • Then select the country or region where the user resides.
  • When finished, press 'OK'.
