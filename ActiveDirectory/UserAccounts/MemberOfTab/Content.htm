The 'Member Of' tab lists all the groups this user is a member of.
  • If you want to add this user to a group, first click on the 'Add...' button.
  • And enter in the group you want this user to be a member of.

  • Then press 'OK'.
  • Once there is more than one group that a user is a member of, you can select a group, which in this case we will select the new group we just added...
  • By selecting a group, it will enable the 'Set Primary Group' button. When this button is pressed, it will configure the primary group that is associated with this user. In a 'Windows' enviroment, this does not affect anything. In other operating systems like Mac, this has an affect on file activity.
  • Press 'OK' when finished.
