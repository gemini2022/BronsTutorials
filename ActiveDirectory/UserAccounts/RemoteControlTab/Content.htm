The 'Remote Control' tab refers to remote control options for 'Remote Desktop' sessions.
  • If you want to disable the remote controll ability for the administrator, you can disable it here.
  • As you can see, by deselecting this option, it eliminates remote controll abilities for the administrator. Therefore, we will reselect this option and move on.
  • Here you have the option to set whether this user needs to give an administrator permissions to view their session.
  • In this last section, you can configure if an administrator will get full control or just view only access to the session.

  • Press 'OK' when finished.
