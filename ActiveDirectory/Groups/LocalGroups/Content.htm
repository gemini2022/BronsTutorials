See the following to learn about local groups...
  • Local groups can have the following members, as long as they belong in the same domain as the local group:
    • Users
    • Computers
    • Domain Local Groups
    • Global Groups
    • Universal Groups
  • Local groups can have the following members, as long as they belong in the same forest as the local group. In other words, the members can be from another domain, but the domain has to be within the same forest:
    • Users
    • Computers
    • Global Groups
    • Universal Groups
  • Local groups can only have the following members if they belong to a domain that's from an external forest:
    • Users
    • Computers
    • Global Groups
