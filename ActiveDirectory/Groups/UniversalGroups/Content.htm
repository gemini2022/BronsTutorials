See the following to learn about universal groups...
  • Universal groups can have the following members, as long as they belong in the same domain as the universal group:
    • Users
    • Computers
    • Global Groups
    • Universal Groups
  • Universal groups can have the following members, as long as they belong in the same forest as the universal group. In other words, the members can be from another domain, but the domain has to be within the same forest:
    • Users
    • Computers
    • Global Groups
    • Universal Groups
  • Universal groups can not have any members if those members belong to a domain that's from an external forest.
