On the 'General' tab, you can view information about the computer account to help identify the account.
  • The first section of the 'General' tab displays the (pre-Windows 2000) computer name.
  • The next section displays the DNS name of the computer account.
  • This section displays the type of computer account this computer is.
  • This next section displays the site this computer account is in.
  • In the last section, you can give a brief description of the computer. This helps other administrators determine what this computer may be used for.
