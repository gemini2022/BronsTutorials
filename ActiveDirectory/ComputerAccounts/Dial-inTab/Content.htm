On the 'Dial-in' tab, except for the first section on this tab, all the other options appy to dial up access to the network using a modem.
  • In the first section, the first two settings allow or deny access. By default, the third option 'Control access through NPS Network Policy' will be checked. Originally, all remote access had to be configured per computer by selecting the first two options either allow or deny. In a large enviroment this becomes very difficult to manage. Also using this approach, management can not be simplified by using groups. With Windows server 2008, Microsoft added the 'Network Policy Server', which allows remote administration to be centralized and conrolled a lot easier.
  • This setting allows you to configure a caller ID. This is used by dial up to confirm that the phone number that is used to connect to the network is the phone number entered here. To configure this option, first enable the 'Verify Caller-ID' option.
  • Then simply enter in the phone number that will be verified with.
  • In this next section are some options relating to callback. Callback means that the server will call the user back on the number given by them or a pre-entered number. To configure this option, first enable the 'Always Callback to' option.
  • Then simply enter in the phone number that will be used to call back.
  • Here is a static IP address that can be configured so that the computer account will always get the same IP address when they connect. To configure this option, first enable the 'Assign Static IP Addresses' option.
  • Then select either the IPv4 option or the IPv6 option. In this case we're going to select the IPv4 option.
  • Then enter in the IP address.

  • And press 'OK'.
  • Here, static routes can be defined. To configure this option, first enable the 'Apply Static Routes' option.
  • Then click on the 'Add Route...' button.
  • Enter in the IP route.
  • Then the network mask.
  • Leave the 'Cost Metric' as 1.

  • Then press 'OK'.
  • When finished, press the 'Ok' button.
  • When finished, press the 'Ok' button.
