Pre-staging a computer is when the domain or enterprise administrator creates a client computer account within the Active Directory domain controller before the client computer has been actually added to the domain. The benefit of pre-staging is that administrators can pre-stage the account in any desired container. When a client computer account is pre-staged, and when an authorized person adds the client computer to the domain, the client computer account is automatically mapped with the pre-staged computer account in that particular container where it was previously created by the administrator. To learn how to pre-stage a computer, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then select 'Active Directory Users and Computers'...
  • Once in 'Active Directory Users and Computers', expand your domain, in this case it's 'bronstutorials.local'.
  • In this example you will see an OU called 'New York' that we have created earlier in another tutorial. For this demonstration we will expand the 'New York' OU.
  • Inside the 'New York' OU, you will notice a child OU that was also created earlier called 'Users'. We want to create another OU next to it inside the 'New York' OU and call it 'Computers'. This is where we will pre-stage a computer. To do this, we'll right-click the 'New York' OU...
  • Select 'New'.
  • And then select 'Organizational Unit'.
  • As mentioned before, we will name this OU 'Computers'.

  • Then click 'OK'.
  • Now that our 'Computers' OU is created, we will pre-stage our computer in it. To do this, right-click on the 'Computers' OU...

 
  • Select 'New'.
  • And then 'Computer'.
  • Here, all you need to do is enter in the name of the computer for the computer account. The name you enter will need to match the computer name of the computer that you are adding to the domain.
  • Next, if we click on the 'Change...' button...
  • We can then set which user or group will be able to add this computer to the domain. As long as the user or group you specify adds this computer to the domain using the name you just gave to this computer account, they will have no problem doing so, no matter what their credentials are. In this case, we're going to allow any user in the 'Help Desk' group permission to add this computer to the domain.

  • After you finish entering in the user or group you want to give permissions to add this computer to the domain, press 'OK'.
  • And then finally, press the 'OK' button to pre-stage this computer.
  • Now that the computer is pre-staged, the authorized person can now follow the same procedure of joining a computer to the domain, but instead of the computer being placed in the default 'Computers' OU, it will automatically be placed in the OU you specified earlier.
