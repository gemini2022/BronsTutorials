On the 'General' tab, you can configure some information about the user's account to help identify the account.
  • As you can see, the first section of the 'General' tab has already been filled in. This is the information you filled in when you first created this user.
  • Here you can add a description of the account. If the account is a service account, it is recommended that you set a description so another administrator will know exactly what the service account is being used for. Having a description lets another administrator know what effect changing properties in the account may have.
  • Here you can enter in the office the account is in.
  • This next section allows for additional information about the user. Here we can enter in the user's phone information.
