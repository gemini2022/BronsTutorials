On the 'Delegation' tab, you can configure this computer account to service delegation.
  • The first option sets this computer account to NOT be trusted for delegation.
  • The second option sets this computer account to be trusted for delegation for any service, but only if Kerberos is being used.
  • The third option sets this computer account to be trusted for delegation for a specified service. But then also allows you to make a choice of which authentication protocol you wish to use (Kerberos only or any authentication protocol).
  • Here we will select the 'Use any authentication protocol' option.
  • And then we are going to click on the 'Add...' button to add the service we want to delegate.
  • Here on this form, click the 'Users or Computers...' button.
  • Then type in the computer where the services resides that we want to delegate.

  • Then press 'OK'.
  • After you press 'OK' on the previous form, you will get a list of all the services of the computer you specified.
  • Then select the service you want to delegate. In this case, we're going to select the DNS service.

  • Then press 'OK'.
  • Press 'OK' when finished.
