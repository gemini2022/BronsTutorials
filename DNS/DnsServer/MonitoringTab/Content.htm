With the Monitoring tab you can perform the DNS lookup test that will run queries against your DNS server. You can also configure automated test that will run at a configured time interval to ensure the DNS server is operating correctly:
  • First, select the option 'A simple query against this DNS server'. This option tests name resolution for the zone loaded on this DNS server.
  • Then select the option 'A recursive query to other DNS servers'. This option tests forwarding. If the simple query passes and the recursive query fails, then the problem is with forwarding.
  • Once either of the above options are selected, press the 'Test Now' button.
  • Here is where it will show the results of the test. In this case, both the 'Simple Query' and the 'Recursive Query' passed. However, if you get a failure, a yellow flag will appear on the server icon in the left-hand pane of the DNS management snap-in. Just be aware that the monitor test produces that flag, and will remain there until you run a successful test.
  • Selecting this option, 'Perform automatic testing at the following interval' will allow testing to occur at a constant interval that you define.
  • Once that option is selected, press 'Apply'.
  • The test will then automatically perform at the interval that was defined.
