The Event Logging tab allows you to restrict the events written to the DNS Events log file to only errors or to only errors and warnings. It also allows you to disable DNS event logging all together.
  • Selecting this option will record no events to the DNS Events log file.
  • Selecting this option will record only error events to the DNS Events log file.
  • Selecting this option will record only error and warning events to the DNS Events log file.
  • This option, the default option, will record all events to the DNS Events log file.
