On the Interfaces tab, you can configure DNS Server to listen on specific interfaces/IP Addresses or all IP addresses. If the server has multiple interfaces then you can configure DNS Server to listen on specific interface. If the server is having only single interface with multiple IP addresses configured then you can configure it to listen on specific IP addresses. By default it is configured to listen on all interfaces and all IP addresses.
  • This option allows DNS Server to listen on all IP addresses.
  • This option allows DNS Server to listen on only specific interfaces/IP Addresses.
