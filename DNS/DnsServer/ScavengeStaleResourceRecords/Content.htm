You can use this procedure to immediately remove server resource records that have exceeded the allotted time since they were created. To learn how to do this, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DNS'...
  • Once in DNS, right-click your server, in this case it is 'DC1'.
  • Then select 'Scavenge Stale Resource Records'.
  • Once you press 'Yes', it is important to understand that this only starts the process up that checks for DNS records that need to be removed. If a zone is not configured for scavenging or a DNS record is not set to be deleted when the DNS record becomes stale, then no DNS records will be removed.
