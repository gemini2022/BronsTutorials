To learn how to view TCP/IP information, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'VPNSERVER (local)'.
  • Then expand 'IPv4'.
  • Once you expand 'IPv4', right-click 'General'..
  • Then select 'Show TCP/IP Information'.
  • This display shows a broad general selection of IP routing data, including the number of routes in the route table, the nuber of IP and UDP datagrams received and forwarded, and the number of connection attempts. To customize the view, you can right-click the TCP/IP information window.
  • And select the option 'Select Columns...'.
  • Once the option 'Select Columns..' is selected, a dialog box will appear. Here you can add and remove parameters that you want displayed in the 'TCP/IP Information' window. For this example, we will add a parameter to the 'TCP/IP Information' window that is currently hidden from the display. In this case, we will select the parameter 'IP incoming header errors'.
  • Then press the 'Add ->' button.
  • Now that the parameter is added to the 'Displayed parameters' list, press 'OK'.
  • A new parameter is now added to the 'TCP/IP Information' window.
