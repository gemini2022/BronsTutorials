To learn about the general tab, proceed with the following steps...
  • This first option, 'Log errors only', specifies that only errors are logged in the System Log in Event Viewer.
  • This next option, 'Log errors and warnings', specifies that errors and warnings are logged in the System Log in Event Viewer.
  • Next, this option 'Log the maximum amount of information', specifies that the maximum amount of information is logged in the System Log in Event Viewer.
  • And then finally, the last option 'Disable event logging', specifies that no events are logged in the System Log in Event Viewer.
