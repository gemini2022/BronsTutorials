Network address translation (NAT) allows you to share a connection to the public Internet through a single interface with a single public IP address. The computers on the private network use private, non-routable addresses. NAT maps the private addresses to the public address. To learn about NAT, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'VPNSERVER (local)'.
  • Then expand 'IPv4'.
  • Once you expand 'IPv4', select 'NAT'.
  • Once you select 'NAT', in the window to the right side of the Routing and Remote Access screen, you will see a list of details that pertain to NAT.
