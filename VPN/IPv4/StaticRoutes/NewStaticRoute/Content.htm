To learn how to create a new static route, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'Routing and Remote Access'...
  • Once in Routing and Remote Acces, expand the server node, which in this case is 'VPNSERVER (local)'.
  • Then expand 'IPv4'.
  • Once you expand 'IPv4', right-click 'Static Routes'.
  • Then select the option 'New Static Route...'.
  • First we have to choose the network interface that this new static route will be configured for. To do this, select the dropdown.
  • And select the network interface you want use with the new static route. In this case we will select 'Local Area Connection'.
  • Next, we have to enter in the destination network that the static route will route to. In this case, the destination network is '192.168.100.0'.
  • And then enter in the network mask of the destination network.
  • Next, we need to type the address of the router that serves as the next hop for forwarded packets.
  • And then finally, type the number that reflects the cost of using this route. Typically, this value is the number of routers between this server and the destination network. When RRAS must select between multiple valid routes, the route with the lowest metric is the one selected.

  • Press 'OK' to create the static route.
