To learn about the Logging tab, proceed with the following steps...
  • This first option specifies that only errors are logged in the System Log in Event Viewer.
  • This next option specifies that errors and warnings are logged in the System Log in Event Viewer.
  • This next option specifies that the maximum amount of information is logged in the System Log in Event Viewer.
  • And this option specifies that no events are logged in the System Log in Event Viewer.
  • And then finally, this last option specifies whether the events in the Point-to-Point Protocol (PPP) connection establishment process for remote access and demand-dial routing connections are written to the PPP.LOG file stored in the systemroot\Tracing folder (the default location).
