To learn how to backup your DHCP server, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, select your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Once your DHCP server has been selected, right-click it...
  • Then select 'Backup...'.
  • From here, you'll be required to select the folder where the DHCP server should place the backed up files. In this case, we will leave it on the default selected folder 'backup'.

  • Then press 'OK' to backup the DHCP server.
