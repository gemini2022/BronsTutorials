To learn how to authorize or unauthorize your DHCP server, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, select your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Once your DHCP server has been selected, right-click it...
  • If your DHCP server is currently authorized, then you will have the option to unauthorize your DHCP server by simply selecting 'Unauthorize'.
  • You will then receive a message warning you that this DHCP server will stop responding to client request if you remove its authorization.

  • Press 'Yes'.
  • This DHCP server is now unauthorized and will no longer respond to client request. However, if we go back and right-click on our DHCP server again
  • We now have the option to authorize our DHCP server by selecting 'Authorize'.
