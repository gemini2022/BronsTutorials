To learn how to restore your DHCP server, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, select your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Once your DHCP server has been selected, right-click it...
  • Then select 'Restore...'.
  • From here, you'll be required to select the folder where the DHCP server can find the backed up files. In this case, we will leave it on the default selected folder 'backup'.

  • Press 'OK'.
  • After pressing 'OK' on the previous form, you will receive a message informing you that the DHCP Server service must be stopped and restarted in order for changes to take effect.

  • Press 'Yes'.
  • After pressing 'Yes' on the previous form, the DHCP Server service will stop.
  • Then it will restart.
  • Once the DHCP database has been successfully restored, press 'OK'.
