A DHCP reservation is a permanent IP address assignment. It is a specific IP address within a DHCP scope that is permanently reserved for leased use to a specific DHCP client. You can configure a DHCP reservation in the DHCP server when you need to reserve a permanent IP address assignment. Reservations are used for DHCP enabled devices like print and file servers or other application servers that always have the fixed IP address on the network. To learn how to configure a DHCP reservation, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, expand your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Then expand 'IPv6'.
  • And then expand 'Scope'.
  • Once 'Scope' has been expanded, select 'Reservations'.
  • Now that you have 'Reservations' selected, right-click it.
  • Then select 'New Reservation...'.
  • First, we need to enter in the name of the reservation. In this case, we'll name this reservation 'Windows 7 Client'.
  • Then enter in the IPv6 address you want to reserve. You are free to enter in any IP address here. The IP address may be in a scope or exclusion range. For this reservation, we'll use the IP address of 'fd00::10'.
  • In order to complte the reservation, you need the 'DUID' and the 'IAID' of the client computer. A good way to get this is...
  • First, go to the client computer you want to reverve the IPv6 address for and select 'Start' or the 'Start Orb' at the bottom left corner of your screen.
  • And type 'cmd' in the search field.
  • Then select the 'cmd' icon to open the command prompt.
  • Once the command prompt has been opened, type in 'ipconfig'.
  • Followed by '/all'.
  • And then finally, type '| clip'. This will output the results to the clipboard.

  • Press 'Enter'.
  • After the command has been run, close the command prompt.
  • Then, go back down to 'Start' or the 'Start Orb' at the bottom left corner of your screen.
  • And this time, type in 'notepad'.
  • Then select the 'Notepad' icon to open Notepad
  • Once 'Notepad' has been opened, paste the contents of the clipboard into it.
  • Here you'll find the information required for the reservation.
  • Now, back at the 'New Reservation' screen, enter in the 'DUID'.
  • Then enter in the 'IAID'.
  • Here you can enter in a description if you wish. In this case, we will enter 'Windows 7 Client'.
  • And then finally, press the 'Add' button.
  • The reservation has now been added. This workstation will now always get the same IP address.
