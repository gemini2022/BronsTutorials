To learn how to create an IPv6 scope, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, expand your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Then select 'IPv6'.
  • Once 'IPv6' has been selected, right-click it...
  • And then select 'New Scope...' to launch the 'New Scope Wizard'.
  • The first screen of the wizard is the welcome screen. Press 'Next' to continue.
  • On this screen, we need to enter in the name of the scope. In this case we're going to name the scope, 'IPv6 Pool'.
  • You can also add a description if you wish. This field is optional, but for this example, we'll enter 'IPv6 General Pool'.

  • Press 'Next >'.
  • On this screen, we need to enter in the corresponding prefix for the IPv6 network.
  • If you have multiple DHCPv6 servers, the preference value can be modified to indicate your priority among the servers. The lower this value, the higher the priority.

  • Press 'Next >'.
  • On this screen, enter any IPv6 address that belongs to that scope but has been manually assigned to other devices in the network. This includes the IPv6 address that is manually configured on the DHCPv6 server itself. Additional exclusion can be added after the initial DHCPv6 scope has been configured. First enter in the start IPv6 address.
  • Then enter in the end IPv6 address.
  • And press 'Add'.
  • After pressing 'Add', the exclued range will be displayed here. You may add more excluded ranges if you wish.

  • Press 'Next >'.
  • On this screen, there are two scope lease settings that need to be configured. The first setting is 'Preferred Life Time'. This setting is the length of time that a valid IPv6 address is preferred. When this time expires, the address becomes deprecated but it is still valid.
  • The next setting is 'Valid Life Time'. This setting is the length of time that an IPv6 is in the valid state. The address becomes invalid after the valid life time expires. The valid life time must be equal or greater than the preferred life time.

  • Press 'Next >'.
  • Finally, press 'Finish' to create the new scope.
