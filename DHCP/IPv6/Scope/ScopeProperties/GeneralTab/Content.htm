To learn about the General tab, proceed with the following steps...
  • Here you have the option of updating the name of the scope.
  • The prefix is the leftmost fields of the IPv6 address. The prefixes in IPv6 can be considered similar to the subnet mask used in IPv4 addresses, the rest specify particular addresses in that network. Thus all the addresses in one network have the same first N bits. Those first N bits are called the "prefix".
  • If you have multiple DHCPv6 servers, the preference value can be modified to indicate your priority among the servers. The lower this value, the higher the priority.
  • The start and end IPv6 addresses of the scope are displayed here.
  • The description for the scope can be edited here.
