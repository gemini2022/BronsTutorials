To learn about the General tab, proceed with the following steps...
  • Here you have the option of updating the name of the scope.
  • Here you can update the starting IP address of the scope.
  • And update the end IP addresss of the scope.
  • The Subnet Mask can be updated if needed
  • The lease duration can be updated here.
  • Or you can set the lease to have no limit.
  • The description for the scope can be edited here.
