A superscope allows a DHCP server to provide leases from more than one scope to clients on a single physical network. Before you can create a superscope, you must use DHCP Manager to define all scopes to be included in the superscope. Scopes added to a superscope are called member scopes. To learn how to create a superscope, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, expand your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Then select 'IPv4'.
  • Once 'IPv4' has been selected, right-click it...
  • And then select 'New Superscope...' to launch the 'New Superscope Wizard'.
  • The first screen of the wizard is the welcome screen. Press 'Next' to continue.
  • On this screen, we need to enter in the name of the superscope. In this case we're going to name the superscope, 'MySuperscope'.
  • On this screen, select the scopes you wish to add to the superscope. In this case we only have one scope to select.
  • Once the scope or scopes have been selected, press 'Next >'.
  • And then finally, press 'Finish' to create the superscope.
