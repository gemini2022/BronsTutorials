To learn how to create an IPv4 scope, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, expand your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Then select 'IPv4'.
  • Once 'IPv4' has been selected, right-click it...
  • And then select 'New Scope...' to launch the 'New Scope Wizard'.
  • The first screen of the wizard is the welcome screen. Press 'Next' to continue.
  • On this screen, we need to enter in the name of the scope. In this case we're going to name the scope, 'IPv4 Pool'.
  • You can also add a description if you wish. This field is optional, but for this example, we'll enter 'IPv4 General Pool'.

  • Press 'Next >'.
  • On this screen, we need to enter the start IP address and the end IP address that we want to allocate in this scope. For the start IP address, we are going to enter '192.168.0.1'.
  • After the start IP address has been entered, the wizard will automatically determine the length of the network portion of the address and the subnet mask based on the IP address that you have entered.
  • Then enter in the end IP address. For the end IP address, we are going to enter '192.168.0.100'.

  • Press 'Next >'.
  • On this next screen, you have the option of entering in exclusions. Any IP addresses you set in this range will NOT be allocated to any clients. In other words, think of the scope as a pool of IP addresses and you are removing some of them from the pool. The rest of the pool will dynamically be allocated to the clients as needed. In this case, we will exclude the first ten IP address from the pool. The start IP address of the range of exclueded IP address will be '192.168.0.1'.
  • And the end IP address will be '192.168.0.10'.
  • Once the start IP address and the end IP address of the excluded range has been entered, press 'Add'.
  • After pressing 'Add', the exclued range will be displayed here. You may add more excluded ranges if you wish.

  • Press 'Next >'.
  • On this screen, you can set how long the client will have the IP address before it will lose it. The client will attempt to renew the lease of the IP address before this time ends. In this case, eight days. If the client is successful of renewing the lease before this time is up, then the client will be able to use the lease for another eight days. As long as the client keeps renewing the lease before it expires, it can use the same IP address. If the lease does expire before the client renews the lease, the DHCP server may allocate the IP address to another client, before the client asks for it back. If this happens, then the client will simply get another IP address from the pool.

  • Press 'Next >'.
  • On this next screen, you have the option to configure common DHCP options for your new scope. You can do this manually at a later time, but it is easier to do it now using the wizard. To do this, leave the default option of 'Yes, I want to configure these options now' selected.

  • Press 'Next >'.
  • The first common option is the default gateway for your network. In this case, the default gateway or router has the address of '192.168.0.1'.
  • Once the default gateway IP address has been entered, press 'Add'.
  • After pressing the 'Add' button, the IP address of the default gateway will be displayed here.

  • Press 'Next >'.
 
  • On this screen, you can enter in the default domain suffix for your network. In most cases, this will already be populated for you.
  • In this section, your DNS server(s) may also already be populated. If not, you can enter them here.

  • Press 'Next >'.
  • On this screen, you can enter in the IP address of your WINS server if you have one. In this case, there is no WINS server on this network, therefore there is no need to configure anything on this screen.

  • Press 'Next >'.
  • On this screen, you have the option of activating this scope now. All scopes must be activated before they can start to issue out IP addresses. In this case, we will go ahead and keep the default option 'Yes, I want to activate this scope now' selected.

  • Press 'Next >'.
  • Finally, press 'Finish' to create the new scope.
