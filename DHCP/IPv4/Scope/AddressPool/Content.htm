To learn about the Address Pool folder, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, expand your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Then expand 'IPv4'.
  • And then expand 'Scope'.
  • Once 'Scope' has been expanded, select 'Address Pool'.
  • After you have selected 'Address Pool', look to the far right of the DHCP console window. Here you can see which IP addresses are available for clients.
  • And which IP addresses have been excluded.
  • However, you have the option of adding more exclusion ranges if needed. To do this, go back to the left side of the DHCP console window and right-click on 'Address Pool'.
  • Then select 'New Exclusion Range...'.
  • Begin by entering in the start IP address of the exclusion range.
  • Then enter the end IP address of the exclusion range.
  • Once the start and end IP addresses of the exclusion range have been entered, press 'Add'.
  • The exclusion range has now been added. You may enter more exclusion ranges if needed, otherwise press 'Close'.
