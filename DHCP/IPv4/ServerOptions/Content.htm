Server options are settings that will be applied to all IP addresses allocated from this DHCP server. To learn how to configure server options, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, expand your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Then expand 'IPv4'.
  • Once 'IPv4' has been expanded, select 'Server Options'.
  • After you have selected 'Server Options', look to the far right of the DHCP console window. There you will notice that there are two settings there already. These are the settings that were configured in the setup wizard. Since these are server options, these settings will be applied to all IP addresses allocated from this DHCP server.
  • However, you have the ability to add additional options. You can do this by right-clicking 'Server Options'.
  • And then selecting 'Configure Options...'.
  • Here you can see, there are a lot of options you can add. Just select the one you want, for example, WINS.

  • Press 'OK'
