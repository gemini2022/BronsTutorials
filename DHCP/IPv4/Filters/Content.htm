It may sometimes be desirable to restrict the range of devices which are allowed to lease IP addresses from a DHCP server. For example, if your environment includes a wireless network, you may wish to allow only certain WiFi-enabled devices to obtain an address. The Windows DHCP service provides MAC address filters for this purpose. To learn how to configure filters, proceed with the following steps...
  • Begin first by selecting 'Start' at the bottom left corner of your screen.
  • Then go to 'Administrative Tools'...
  • Then 'DHCP'...
  • Once in DHCP, expand your DHCP server, in this case the DHCP server is 'dc1.bronstutorials.local'.
  • Then expand 'IPv4'.
  • Once 'IPv4' has been expanded, expand 'Filters'.
  • With filtering, you have the option to allow or deny a client from leasing an IP address based on its MAC address. In this case, we're going to deny a client from leasing an IP address. To do this, select 'Deny'.
  • Then righ-click it...
  • And select 'New Filter...'.
  • In order for the filtering to work, you need the MAC address of client. A good way to get this MAC address is...
  • First, select 'Start' at the bottom left corner of your screen.
  • Then open the command prompt.
  • Once the command prompt is open, type the command 'getmac'.
  • Then type '/s'. This is a parameter that indicates you want to get the MAC address from a remote computer rather than this local computer.
  • Next, enter in the IP address or name of the computer you want to get the MAC address from. In this case, we'll enter in the name of the computer 'ws1'.
  • And then finally, type '| clip'. This will output the results to the clipboard.

  • Press 'Enter'.
  • After the command has been run, close the command prompt.
  • Then go back down to 'Start'....
  • And this time select 'Notepad'.
  • Once 'Notepad' has been opened, paste the contents of the clipboard into it.
  • From here, select and copy the MAC address.
  • Now, back at the 'New Filter' screen, paste the MAC address you just coppied into the 'MAC address' field.
  • Then enter in a description if you wish. In this case, we'll enter 'Deny WS1'.
  • Once the MAC address and an optional description has been entered, press 'Add'.
  • You can add more if you wish, otherwise press 'Close'.
  • In order for this deny filtering to work, you must first enable it. Do do this, right-click it...
  • Then press 'Enable'.
  • You’d think that after you click 'Enable' it will enable MAC address filtering for your allow and deny lists. However, there is one more thing you need to do. Right-click 'IPv4'...
  • Then select 'Properties'.
  • Once the properties have been opened, select the 'Filters' tab.
  • With this option selected, the allow list, when configured, will be enabled.
  • With this option selected, the deny list, when configured, will be enabled.
  • If we select the 'Advanced...' button.
  • You can configure which hardware types will be exempted from filtering.

  • Press 'OK'.
